1955 年。 那年 5 月， 《美国数学月刊》 (American Mathematical Monthly) 刊登了一篇题为 “论素数的无穷性” (On the Infinitude of Primes) 的论文， 作者是美国耶什华大学 (Yeshiva University) 的德裔本科生弗斯滕伯格 (Hillel Furstenberg)。

在这篇论文中， 弗斯滕伯格定义了整数集上的一个拓扑， 其开集由且仅由空集 ∅ 及算术序列 (arithmetic progression) aℤ + b (a ≠ 0 和 b 皆为整数) 的并集组成[注九]。 不难证明， 如此定义的开集满足拓扑的定义， 即：

空集和全集是开集 (前者由定义直接给出， 后者是定义中 a = 1, b = 0 的特例)；
任意开集的并集是开集 (由定义直接给出)；
任意有限多个开集的交集是开集 (可证明如下： 设正整数 x 是有限交集 ∩Ui 的元素， 则对每个 Ui 存在 ai， 使 x ∈ aiℤ + x ⊂ Ui。 设 a 为所有 ai 的最小公倍数， 则显然对所有 i 皆有 x ∈ aℤ + x ⊂ aiℤ + x ⊂ Ui， 因此 x ∈ aℤ + x ⊂ ∩Ui。 这说明 ∩Ui 若包含元素 x 就必然包含它的一个开邻域 aℤ + x， 因此 ∩Ui 是开集， 证毕。 感兴趣的读者请想一想， “有限” 二字体现在哪里？)。
弗斯滕伯格的拓扑有两个重要特点：

所有非空开集都包含无穷多个元素 (因为是无穷集合 aℤ + b 的并集)， 因此任何非空有限集都不可能是开集；
所有形如 aℤ + b 的集合都是既开且闭的——其中 “开” 由定义可知， “闭” 则是因为 aℤ + b 与 (aℤ + b + 1)∪···∪(aℤ + b + a — 1) 恰好包含全部整数 ℤ， 因此互为补集， 但 (aℤ + b + 1)∪···∪(aℤ + b + a — 1) 按定义乃是开集， 因此其补集 aℤ + b 是闭集。
有了这些铺垫， 现在可以完成证明了， 由于 pℤ 对所有素数取并集所得的 ∪(pℤ) 包含了除 {—1, 1} 以外的全部整数 (因为除 {—1, 1} 外的全部整数都有素因子分解)， 因此 {—1, 1} 与 ∪(pℤ) 互为补集。 由于 pℤ 按上述第 2 个特点是闭集， 因此假如素数只有有限多个， 则 ∪(pℤ) 也是闭集 (因为由拓扑定义的第 3 条——即任意有限多个开集的交集是开集——可推知任意有限多个闭集的并集是闭集)， 其补集 {—1, 1} 则是开集。 但这跟上述第 1 个特点——即任何非空有限集都不可能是开集——相矛盾。 因此素数有无穷多个。

给出上述视角别致的精彩证明时， 弗斯滕伯格才 20 岁， 他那篇论文只有 1/3 页， 正文仅 12 行 (上文由于作了详细说明， 篇幅显著大于原文)， 却堪称一鸣惊人。
