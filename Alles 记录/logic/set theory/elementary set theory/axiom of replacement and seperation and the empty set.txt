事实上要不要空集公理取决于你的replacement的formulation，如果你要求公式定义出来的类F是个全函数，那么需要空集公理；但如果你只需要F在A（分离公理：Given any set A, there is a set B (a subset of A) such that, given any set x, x is a member of B if and only if x is a member of A and φ holds for x）上具有“唯一值性”（例如这样的formulation：for every formula φ(u,v,p1,…,pn), fix the parameters p1,…,pn and pick any set A, whenever u∈A has at most one v for which φ(u,v,p1,…,pn) is true, then the collection of {v∣φ(u,v,p1,…,pn),u∈A} is also a set.），那么不需要空集公理
