
一位逻辑学家误入某部落，结果被囚于牢房之中，有一天，酋长对逻辑学家说：“你的牢房有两个门,其中有一个是通向自由的‘自由之门’，而另一个是通向死亡的‘死亡之门’。你可以通过开启一门，从而来决定你自己的生死命运。现在，为了让你能够判断哪一扇门才是‘自由之门’，我特意安排了两个战士来帮助你。但是，这两个战士中，有一人天性诚实，他永远都说真话，另一人则撤谎成性，他永远都说假话。此刻你只能向他们中的一人提出一个你想要问的问题，而且，他们也只能用‘是’或‘否’来作答。好了，你的生死就交由你自己了，请发问吧。”
逻辑学家思考了片刻，随即指着一扇门，问其中一个战士：“如果我问另一个战士这扇门是不是自由之门，他会回答‘是’吗？”。在得到回答之后，他准确地打开了自由之门，安全地离去。

“這個試題的問題為：Cheryl（C）同時向2位新朋友Albert（A）和Bernard（B）提供了10個日期的選項，並分別向A和B說出真正生日的月份和日子（即A知道月份，B知道日子），讓他們各自猜出她自己的真正生日日期。
10個選項為：
5月15日；5月16日；5月19日；6月17日；6月18日；7月14日；7月16日；8月14日；8月15日；8月17日。
另外，此題目也給出了3個說明：

2. B：我本來不知道C的生日，但現在知道了。
3. A：那么我也知道真正生日了。”

一个老题： （假设A、B、C三人的逻辑推理能力都是无限强的）C 随机产生了两个不同的正整数，分别交给了 A 、 B ，并让两人猜测谁手中的数更大。然后 A 说不知道， B 说不知道， A 说还是不知道， B 也说还是不知道，然后 A 说知道了， B 说不但知道了，而且这两个数具体是多少都知道了。问这两个数是多少。

一个来自hamkins的新题目（https://douc.cc/29dLVr）：
A 、 B 两人在主持人 C 的带领下玩一个游戏。 C 向两人宣布游戏规则：“一会儿我会随机产生两个不同的形如n - (1/2)^k - (1/2)^k+r的数，其中 n 、 k 是正整数， r 是非负整数。然后，我会把这两个数分别交给你们。你们每个人都只知道自己手中的数是多少，但不知道对方手中的数是多少。你们需要猜测，谁手中的数更大一些。”这里，我们假设所有人的逻辑推理能力都是无限强的，并且这一点本身也成为了共识。 C 按照规则随机产生了两个数，把它们交给了 A 和 B ，然后问他们是否知道谁手中的数更大。于是有了这样的一段对话。
A ：我不知道。
B ：我也不知道。
A ：我还是不知道。
B ：我也还是不知道。
C ：这样下去是没有用的！可以告诉你们，不管你们像这样来来回回说多少轮，你们仍然都没法知道，谁手中的数更大一些。
A ：哇，这个信息量好像有点儿大！不过，即使知道了这一点，我还是不知道谁手中的数更大。
B ：我也还是不知道。
A ：我继续不知道。
B ：我也继续不知道。
C ：还是套用刚才的话，不管你们像这样继续说多少轮，你们仍然没法知道谁手中的数更大。
A ：哦……不过，我还是不知道谁手中的数更大。
B ：而且我也还是不知道。我们究竟什么时候才能知道呢？
C ：事实上啊，如果我们三个就像这样继续重复刚才的一切——你们俩互相说一堆不知道，我告诉你们这样永远没用，然后你们继续互说不知道，我继续说这不管用——那么不管这一切重复多少次，你们仍然不知道谁手中的数更大！
A ：哇，这次的信息量就真的大了。只可惜，我还是不知道谁的数更大一些。
B ：我也还是不知道。
A ：是吗？好，那我现在终于知道谁的数更大了。
B ：这样的话，那我也知道了。而且，我还知道我们俩手中的数具体是多少了。
A ：那我也知道了。

那么 ， C 究竟把哪两个数给了 A 和 B ？
