Roughly speaking, an indiscernible sequence is an infinite sequence of elements which is very homogeneous, in the sense that

  • Any two elements have the same type
  • More generally, any two subsequences of the same length have the same type.

Definitions[]

Fix some structure M. A sequence

a1, a2, …

of elements of M (or more generally, tuples from M) is said to be indiscernible if any two subsequences of the same length have the same type, i.e., if

tp(ai1ai2⋯ain/⌀) = tp(aj1aj2⋯ajn/⌀)

for any

i1 < i2 < ⋯ < in < ω

and

j1 < j2 < ⋯ < jn < ω.

(Here, we are using the standard notational abbreviation in which concatenation of tuples is written multiplicatively.)

More generally, if A ⊆ M is some subset, we say that a1, a2, … is A-indiscernible if it is indiscernible after naming the elements of A, or equivalently, if

tp(ai1ai2⋯ain/A) = tp(aj1aj2⋯ajn/A)

for any

i1 < i2 < ⋯ < in < ω

and

j1 < j2 < ⋯ < jn < ω.

More generally, we can allow the sequence of ai's to have length other than ω. For any infinite totally ordered set (I,  < ), a sequence {ai}i ∈ I is A-indiscernible if

tp(ai1ai2⋯ain/A) = tp(aj1aj2⋯ajn/A)

for any

i1 < i2 < ⋯ < in

and

j1 < j2 < ⋯ < jn

in I. Again, this just means that any two subsequences of the same length have the same type (over A).

Remark: In the case n = 1, the definition of "A-indiscernible" implies that for any i = i1 and any j = j1, tp(ai/A) = tp(aj/A). In other words, any two elements of an indiscernible sequence must have the same type. However, the condition of indiscernibility is much stronger than this. For example, it also implies that

  • aiai + 1≡Aajaj + 1, i.e., the type of any two consecutive pairs is the same.
  • a1a2a3⋯≡Aa2a3a4⋯, i.e., we can shift the entire sequence to the left and its type doesn't change. (To verify this statement, it suffices to show that tp(ai1⋯ain/A) = tp(ai1 + 1⋯ain + 1/A) for each i1 < ⋯ < in. This is clear from the definition of indiscernibility.)
  • a1a2a3⋯≡Aa2a4a6a8⋯.
  • More generally, any infinite subsequence of an indiscernible sequence (of length ω) has the same type as the original sequence. In fact, this condition characterizes indiscernible sequences (of length ω), by an easy exercise.

Examples[]

Example: In the structure (ℚ,  < ) (a model of DLO), any increasing sequence a1 < a2 < ⋯ is indiscernible. In fact, by quantifier elimination in DLO, the type of a tuple (x1, x2, …, xn) over Ø is completely determined by the relative ordering of the xi's. So if i1 < ⋯ < in and j1 < ⋯ < jn, then

tp(ai1ai2⋯ain/⌀) = tp(aj1aj2⋯ajn/⌀)

because both tuples (ai1, ai2, …, ain) and (aj1, aj2, …, ajn) are strictly increasing.

Example: Consider the structure of ℂ as a ring (a model of ACF). Let t1, t2, … be a sequence of transcendental complex numbers which are algebraically independent. (For example, t1, t2, … could be the beginning of an enumeration of a transcendence basis of ℂ over ℚ.) Then t1, t2, … is indiscernible. To see this, let i1 < ⋯ < in and j1 < ⋯ < jn be two increasing sequences. Then ℚ(ti1, …, tin) ⊂ ℂ is isomorphic to the rational function field in n variables, because the ti1, …, tin are algebraically independent. The same is true of ℚ(tj1, …, tjn). Consequently we can get an isomorphism

$f:\mathbb{Q}(t_{i_{1}},t_{i_{2}},\ldots,t_{i_{n}})\overset{\sim}{\rightarrow}\mathbb{Q}(t_{j_{1}},t_{j_{2}},\ldots,t_{j_{n}})$

such that f(tik) = tjk for each k. This condition is equivalent to the assertion that the two tuples in question have the same quantifier free type:

qftp((ti1, …, tin)/⌀) = qftp((tj1, …, tjn)/⌀).

Then because of quantifier elimination in ACF, it follows that they actually have the same type. Consequently the sequence is indiscernible.

Non-example: Let M be the structure (ℤ, s), where s is a unary function symbol interpreted as s(x) = x + 1. The sequence 1, 2, 3, ... is not indiscernible in this structure. Any two elements of this sequence have the same type over Ø, and it is also true that

(1, 2, 3, …)≡⌀(2, 3, 4, …).

Both of these facts hold because s is an automorphism of M. However, indiscernibility fails because, e.g.,

(1, 2)≢⌀(1, 3)

Indeed, the tuple (1,3) satisfies the formula y = s(s(x)), but the tuple (1,2) does not. So they do not have the same type over the empty set, and the sequence is not indiscernible.

Totally Indiscernible Sequences[]

A sequence {ai}i ∈ I is said to be totally indiscernible if

ai1ai2⋯ain≡⌀aj1aj2⋯ajn,

for any i1, …, in, j1, …, jn, subject only to the condition that

ik ≠ ik′ and jk ≠ jk′ for k ≠ k′.

An equivalent condition is that {ai}i ∈ I is indiscernible, and the sequence remains indiscernible after rearranging the order of the terms.

The condition for {ai}i ∈ I to be totally indiscernible doesn't depend on the ordering of I, and is really more of a property of the set {ai : i ∈ I}, so the terminology indiscernible set is sometimes used in stead of "totally indiscernible sequence."

Of course we can also add a parameter set A, and speak of "totally A-indiscernible sequences," or equivalently, "A-indiscernible sets."

Example: The example of an independent sequence of transcendentals t1, t2, … in ℂ is totally indiscernible. Indeed, any reordering of this sequence will still be an independent sequence of transcendentals, and so will remain indiscernible by the argument given above.

More generally, it turns out that in any stable theory, every indiscernible sequence is totally indiscernible. We do not prove this here.

A closely related source of totally indiscernible sequences is the Morley sequence of a generically stable type in an NIP theory.

Non-example: In the structure (ℚ,  ≤ ), the sequence 1, 2, 3, ... is indiscernible, as we discussed above. However, this sequence is not totally indiscernible. For instance, the tuple (1,2) does not have the same type as (2,1), because the former satisfies the formula x < y , while the latter does not. In terms of rearranging the sequence, the sequence fails to be totally indiscernible, because the reordering

2, 1, 3, 4, 5, …

is not indiscernible.

More generally, in a totally ordered structure, every totally indiscernible sequence must be constant. Indeed, if a1, a2, … is totally indiscernible, then

tp(a1a2/⌀) = tp(a2a1/⌀), so

a1 ≤ a2 ⇔ a2 ≤ a1.

This is only possible if a1 = a2. But then indiscernibility gives ai = aj for any i < j, so the sequence must be constant.

Another example of a theory in which totally indiscernible sequences are scarce is the p-adics (the theories of p-closed fields.) For example, in the case of the 2-adics ℚ2, we can argue as follows. First note that there are no equilateral triangles in ℚ2, in the sense that

ℚ2 ⊨ ∀x, y, z : v(x − y) = v(x − z) = v(y − z) → x = y = z

Indeed, looking at the binary expansions of x, y, and z, we can find the rightmost digit (bit) at which any two of x, y, and z differ. Two of x, y and z must agree because there are only two binary digits. If x and y agree, and z differs, this means that v(x − y) > v(x − z) = v(y − z).

More formally, given an equilateral triangle {x, y, z}, we can scale it so that v(x − y) = v(y − z) = v(x − z) = 0 (or in terms of the 2-adic norm, |x − y|2 = |x − z|2 = |y − z|2 = 1. Then each of the three differences is in ℤ2 and has nonzero residue. There is only one nonzero residue, however, so res(x − y) = res(x − z) = res(y − z) = 1. But this is absurd since 1 = res(x − z) = res(x − y) + res(y − z) = 1 + 1, and 1 + 1 is 0, not 1.

So there are no equilateral triangles in ℚ2. This is a first order statement, so if K is any 2-closed field (any valued field elementarily equivalent to ℚ2), then

K ⊨ ∀x, y, z : v(x − y) = v(y − z) = v(x − z) → x = y = z.

Now let x1, x2, … be a totally indiscernible sequence in K. Because the valuation is ultrametric, the triangle {x1, x2, x3} must be isoceles. Because any permutation of the tuple (x1, x2, x3) must have the same type as (x1, x2, x3), the triangle is forced to be equilateral. This then forces x1 = x2 = x3. As in the ordered case, this forces the entire sequence to be constant.

In the case of ℚp for p > 2, one can instead show that

$\mathbb{Q}_{p} \vDash \forall x_{1},\ldots,x_{p + 1}:\left( {\bigwedge\limits_{i < j,k < \ell}v(x_{i} - x_{j}) = v(x_{k} - x_{\ell})} \right)\rightarrow\left( {\bigwedge\limits_{i,j}x_{i} = x_{j}} \right)$

again by looking at the base p expansion and using the pigeonhole principle at the rightmost digit where things begin to differ. This statement must then persist in all p-closed fields, and then one can make a similar argument to show that any totally indiscernible sequence must be constant.

Manipulating Indiscernible Sequences[]

We list some basic facts about indiscernible sequences which are relatively easy to verify from the definitions.

If a1, a2, … is an indiscernible sequence, so is the order-reversed sequence …, a3, a2, a1. This works for A-indiscernible sequences of any length.

If a1, a2, … is an indiscernible sequence, then so is any infinite subsequence ai1, ai2, …, with i1 < i2 < ⋯. In fact, ai1, ai2, … has the same type as the original indiscernible sequence. In general, if {ai}i ∈ I is an A-indiscernible sequence and J ⊂ I is an infinite subset, then {ai}i ∈ J is also A-indiscernible, and if J and I are order-isomorphic, then {ai}i ∈ J has the same type over A as the original indiscernible sequence {ai}i ∈ I:

{ai}i ∈ J≡A{ai}i ∈ I

More precisely, if f : J → I is an order-preserving bijection, then the map sending ai to af(i) and sending A to itself is a partial elementary map.

To check that an infinite sequence is indiscernible, it suffices to check that every countable subsequence is indiscernible.

To check that a sequence is A-indiscernible, it suffices to check that it is A0-indiscernible for every finite subset A0 ⊆ A.

If a1, a2, … is indiscernible, then we can clump consecutive terms together without losing indiscernibility. Specifically, the following sequences are indiscernible:

  • a1a2, a3a4, a5a6, …
  • a1a2a3, a4a5a6, a7a8a9, ….
  • and so on

If f is an A-definable function, and {ai}i ∈ I is A-indiscernible, then so is {f(ai)}i ∈ I.

Any constant sequence a, a, a, … is indiscernible. If a1, a2, … fails to be non-constant, then it must contain no repeats, that is, ai ≠ aj for any i ≠ j. (If not, then ai = aj for some i < j. Then indiscernibility implies that for any i < j, we have ai = aj. This forces the sequence to be constant.)

If {ai}i ∈ I is A-indiscernible, it remains A-indiscernible upon passing to any elementary extension.

Any sequence having the same type over A as an A-indiscernible sequence is itself A-indiscernible. The property of being A-indiscernible is type-definable, in the sense that there is a partial type Σ(x1, x2, …) over A such that a1, a2, … is A-indiscernible if and only if M ⊨ Σ(a1, a2, a3, …). (Specifically, Σ(x1, x2, …) consists of the formulas

ϕ(xi1, …, xin, a) ⇔ ϕ(xj1, …, xjn, a)

for every formula ϕ(z1, …, zn, y), every a from A, and every i1 < i2 < ⋯ < in < ω and j1 < j2 < ⋯ < ω.)

If a sequence is A-indiscernible, it is also B-indiscernible for every B ⊆ A.

If {ai}i ∈ I is A-indiscernible, then it is also acl(A)-indiscernible, where acl(A) denotes the algebraic closure of A. We will prove this later after discussing extraction of indiscernible sequences.

All of the above claims also work for totally indiscernible sequences (rather than indiscernible sequences). In many of the cases, this can be seen by the characterization of totally indiscernible sequences as those sequences which remain indiscernible after re-ordering.

Extending Indiscernible Sequences[]

Let a1, a2, … be an A-indiscernible sequence in the structure M. By passing to an elementary extension, or assuming that M is sufficiently saturated, we can extend this A-indiscernible sequence to an A-indiscernible sequence of length ω + ω

a1, a2, …, aω, aω + 1, ….

In fact, we can extend the sequence to whatever length we like. We can even extend it to the left, or add new elements in between preexisting ones. For example, we can also find a1.5, a2.5, …, such that

a1, a1.5, a2, a2.5, …

is A-indiscernible.

More precisely, we have the following statement:

Theorem: Let J be an infinite totally ordered set, and let I ⊂ J be an infinite subset of J. Let M be a structure, and {ai}i ∈ I be an A-indiscernible sequence indexed by I. Then there is an elementary extension M′ ≽ M in which there is an A-indiscernible sequence {bj}j ∈ J such that bi = ai for i ∈ I.

This can be stated a little more cleanly if we work in a monster. Theorem: Let J be an infinite totally ordered set, and let I ⊂ J be an infinite subset of J. If {ai}i ∈ I is A-indiscernible, then we can find aj for j ∈ J ∖ I such that {aj}j ∈ J is A-indiscernible.

These theorems also hold with indiscernibility replaced by total indiscernibility.

We will prove these claims after discussing extraction of indiscernible sequences.

Extracting Indiscernible Sequences[]

Let {ai}i ∈ I be some infinite sequence of tuples from M, not assumed to have any indiscernibility properties. (But we should probably assume that ai and aj have the same length and live in the same sorts, for i ≠ j.) Let A be some set of parameters. The Ehrenfeucht-Mostowski type of the sequence {ai}i ∈ I over A is the set of all A-formulas

ϕ(x1, …, xn)

such that M ⊨ ϕ(ai1, …, ain) for every i1 < ⋯ < in. We write EM type as shorthand for "Ehrenfeucht-Mostowski type."

For example, if we consider the sequence 0, 1, 2, 3, … in the structure ℝ (as a model of RCF), then the EM type will contain statements like

  • x1 ≥ 0 (because every element of the sequence is at least 0)
  • x2 > x1 (because the sequence is increasing)

But it will not contain, for example

x2 = x1 + 1, because while this holds for consecutive terms, it does not hold for non-consecutive terms.

We can say that some other sequence {bj}j ∈ J "realizes" the EM type of {ai}i ∈ I over A if every formula in the EM type of the latter sequence is also in the EM type of the former sequence.

Observation: An infinite sequence {ai}i ∈ I is A-indiscernible if and only if its EM type is "complete," in the sense that for every formula ϕ(x1, …, xn), either ϕ or ¬ϕ is in the EM-type of {ai}i ∈ I.

Proof: First suppose that {ai}i ∈ I is A-indiscernible. Let ϕ(x1, …, xn) be a formula over A. Let i1 < ⋯ < in be n distinct elements of I, in sorted order. (We can find such ik because I is infinite.) Then either M ⊨ ϕ(ai1, …, ain) or M ⊨ ¬ϕ(ai1, …, ain). If the former happens, then ϕ is in the EM type of the sequence over A, because of A-indiscernibility. If the latter happens, then ¬ϕ is in the EM type over A, again by A-indiscernibility.

Conversely, suppose that {ai}i ∈ I is an infinite sequence with a "complete" EM type. Then for i1 < ⋯ < in and j1 < ⋯ < jn we want to show that

ai1⋯ain≡Aaj1⋯ajn.

But if this failed, there would be an A-formula ϕ(x1, …, xn) such that

M ⊨ ϕ(ai1, …, ain) and M ⊨ ¬ϕ(aj1, …, ajn).

But then ϕ is not in the EM type, because of aj1⋯ajn, and ¬ϕ is not in the EM type, because of ai1⋯ain. QED.

Observation: If {ai}i ∈ I and {bi}i ∈ I are two A-indiscernible sequences indexed by the same ordered set I, then {ai}i ∈ I and {bi}i ∈ I have the same type over A if and only if they have the same Ehrenfeucht-Mostowski type over A.

Indeed, the EM type of an indiscernible sequence completely specifies the type of any subsequence, and these data combine to yield the type of the entire sequence.

At any rate, the fundamental theorem about extracting indiscernible sequences is the following. For simplicity, work in a monster model.

Theorem: Let {ai}i ∈ I be a small infinite sequence of elements of the monster. Let J be some small totally ordered set. Let A be a small subset of the monster. Then we can find an A-indiscernible sequence {bj}j ∈ J which realizes the EM type of {ai}i ∈ I over A.

Without working in the monster, it is necessary to pass to an elementary extension to find such a sequence.

In the case where {ai}i ∈ I is already A-indiscernible, this theorem is an easy application of compactness. Specifically, we write down the (infinitary) partial type $\Sigma(\overset{\rightarrow}{x})$ in the variables {xj}j ∈ J consisting of the statements

ϕ(xj1, …, xjn)

for every j1 < ⋯ < jn ∈ J and every formula ϕ(x1, …, xn) in the EM type (over A) of the original sequence {ai}i ∈ I. If we can realize this type, then we will have found a sequence {bj}j ∈ J whose EM type extends the EM type of the original sequence. But since the EM type of the original sequence was complete, the EM type of the {bj}j ∈ J must also be complete, making this sequence be indiscernible. Then we are done.

If, on the other hand, we cannot realize this partial type $\Sigma(\overset{\rightarrow}{x})$, then by compactness or saturation, there must be some finite subset $\Sigma_{0}(\overset{\rightarrow}{x}) \subset \Sigma(\overset{\rightarrow}{x})$ which is unsatisfiable. This finite subset must only refer to finitely many variable xj1, …, xjN among the ones occurring in $\overset{\rightarrow}{x}$. Write $\Sigma_{0}(\overset{\rightarrow}{x})$ as Σ0(xj1, …, xjN). Since I is infinite, we can find i1 < ⋯ < iN in I. Then

 ⊨ Σ0(ai1, …, aiN)

essentially by definition of Σ. This contradicts the unsatisfiability of Σ0. QED.

The Theorem on extracting indiscernible sequence has a number of useful consequences:

Corollary: Nontrivial indiscernible sequences exist, in infinite structures. More precisely, let M be an infinite structure (or more precisely, a structure containing an infinite sort). Then, in some elementary extension of M, we can find an indiscernible sequence a1, a2, … which is non-constant (i.e., ai ≠ aj for some i and j.)

Proof: Let b1, b2, … be some infinite sequence of distinct elements of M (from the same sort). The EM type of {bi}i ∈ ℕ contains the formula x1 ≠ x2, because no two elements are the same. Let a1, a2, … be an indiscernible sequence extracted from the bi's. Then the EM type of a1, a2, … must also contain the formula x1 ≠ x2. This implies that ai ≠ aj for i < j. QED.

Corollary: We can extend indiscernible sequences. Working in a monster model, let J be a small totally ordered set (an abstract set, not a subset of the monster). Let I ⊆ J be an infinite subset. Then given any indiscernible sequence {ai}i ∈ I, we can choose aj for j ∈ J ∖ I such that {aj}j ∈ J is also indiscernbile. (Of course we can also do this with a parameter set A.)

Proof: Let {bj}j ∈ J be an indiscernible sequence of length J extraced from {ai}i ∈ I. These two sequences have the same EM type (over Ø, or over whatever parameter set we may want to use). Consider the subsequence {bi}i ∈ I of {bj}j ∈ J. Then all three sequences

  • {ai}i ∈ I
  • {bj}j ∈ J
  • {bi}i ∈ I

realize the same EM type. (The EM type of the third sequence must extend the EM type of the second sequence. But since the second sequence already has a complete EM type, the EM type of the third sequence must agree with it.)

Now since the first sequence {ai}i ∈ I has the same EM type and the same length as {bi}i ∈ I, they must have the same type, by one of the observations above.

In particular, by strong homogeneity of the monster, we can find an automorphism σ (over Ø or a parameter set A) such that σ(bi) = ai for each i. Now let aj = σ(bj) for every j ∈ J. Then {aj}j ∈ J is indiscernible, because it is the image of an indiscernible sequence under an automorphism. QED.

We can also extend totally indiscernible sequences, in a manner similar to the Corollary just discussed. This essentially holds because the total indiscernibility is witnessed by the EM type: an A-indiscernible sequence {ai}i ∈ I is totally A-indiscernible if and only if its EM type includes

ϕ(x1, …, xn) ⇔ ϕ(xπ(1), …, xπ(n))

for any A-formula ϕ(x1, …, xn) and any permutation π of {1, …, n}.

Finally, we show as promised that A-indiscernibility implies acl(A)-indiscernibility.

Lemma: Work in a monster. Let {ai}i ∈ I be an infinite sequence (of small length), indiscernible over some small set A. Let D be a non-empty A-definable set. Then there exists some d ∈ D such that {ai}i ∈ I is Ad-indiscernible. (Here, Ad denotes A ∪ {d}, as usual.)

Proof: Let e be any element of D. Let {bi}i ∈ I be an Ae-indiscernible sequence extracted from {ai}i ∈ I. The EM type of {bi}i ∈ I over A must extend the EM type of {ai}i ∈ I over A, which was already complete. So the two sequences {ai}i ∈ I and {bi}i ∈ I have the same type over A (as they have the same EM type and the same length). Let σ be an automorphism of the monster over A, sending bi to ai for each i.

Since {bi}i ∈ I is Ae-indiscernible, {σ(bi)}i ∈ I = {ai}i ∈ I is Aσ(e)-indiscernible. And, since D is A-definable and σ fixes A, the element σ(e) is in D. Take d = σ(e). QED.

Theorem: Work in a monster. Let {ai}i ∈ I be an infinite sequence (of small length), indiscernible over some (small) set A. Then there is a small model M containing A, such that {ai}i ∈ I is indiscernible over A. (Since we are working in the monster, a "model" is a small elementary substructure of the monster.)

Proof: If A is not a model, then by the Tarski-Vaught criterion, there is some non-empty A-definable set D such that no element of D is in A. By the Lemma, we can find some element d ∈ D such that the sequence remains indiscernible over A if we add d to A. By repeating this process infinitely many times, we can enlarge A to be a model.

More rigorously, let {Dα}α < λ be some enumeration of the A-definable sets. Then we can recursively build an increasing sequence Aα of small subsets such that {ai}iinI is Aα-indiscernible for each α ≤ λ, and such that Aα + 1 contains an element of Dα. Specifically, we take

  • A0 = A
  • $A_{\alpha} = \bigcup\limits_{\beta < \alpha}A_{\beta}$ if α is a limit ordinal. The sequence is Aα-indiscernible because it is indiscernible over any finite subset of Aα.
  • Aα + 1 = Aαdα for some dα ∈ Dα, such that the indiscernibility is maintained. We can find such a dα by the Lemma.

Now the limit A′ := Aλ is a small subset with the property that the original sequence {ai}i ∈ I is A′-indiscernible, and every A-definable set contains at least one point in A′. Repeating this process ω times, we obtain

A ⊂ A′ ⊂ A″ ⊂ ⋯ ⊂ A(n) ⊂ ⋯

such that the original sequence {ai}i ∈ I is A(n)-indiscernible for each n, and such that each A(n)-definable set has points in A(n + 1).

Now let M be the union $\bigcup\limits_{n = 1}^{\infty}A^{(n)}$. Again, we have that {ai}i ∈ I is M-indiscernible. And any M-definable set it A(n)-definable, hence has at least one point in M ⊇ A(n). But this latter condition means that M is a model (i.e., an elementary substructure of the monster), by the Tarski-Vaught criterion. QED.

Corollary: Let {ai}i ∈ I be A-indiscernible. Then {ai}i ∈ I is acl(A)-indiscernible.

Proof: We can elementarily embed everything in a monster. Then we are guaranteed some model M ⊇ A such that {ai}i ∈ I is M-indiscernible. But acl(A) is the same computed in our original model as in the monster, as in M. In particular, acl(A) ⊆ M, so {ai}i ∈ I is acl(A)-indiscernible. QED.

Proof of Existence of Indiscernible Sequences[]

Work in a monster model. Assume for the sake of simplicity that our base set of parameters A is the empty set Ø. Given an infinite sequence {ai}i ∈ I, and an ordered set J, we want to find an indiscernible sequence {bj}j ∈ J realizing the EM type of {ai}i ∈ I.

Above, we gave a proof using compactness that worked whenever {ai}i ∈ I was already indiscernible to begin with (so we were really just changing the length of the sequence). The same proof yields

Lemma: Let {ai}i ∈ I be an infinite sequence (of small length). Let J be an ordered set (small). Then we can find {bj}j ∈ J not necessarily indiscernible, but realizing the EM type of {ai}i ∈ I.

Proof: This amounts to realizing the partial type $\Sigma(\overset{\rightarrow}{x})$ consisting of the formulas

ϕ(xj1, …, xjn)

for every j1 < ⋯ < jn ∈ J and every ϕ(x1, …, xn) in the EM type of {ai}i ∈ I. By compactness, it suffices to realize a finite subset $\Sigma_{0}(x_{j_{1}},\ldots,x_{j_{N}}) \subset \Sigma(\overset{\rightarrow}{x})$. But one can easily see that if we take i1 < ⋯ < iNinI, then

 ⊨ Σ0(ai1, …, aiN),

by definition of $\Sigma(\overset{\rightarrow}{x})$ and of the EM type of {ai}i ∈ I. QED.

On account of this lemma, the problem of extracting indiscernible sequences reduces down to the case where the index sets are always ω. That is, we only need to extract an indiscernible sequence b1, b2, … from an (arbitrary) sequence a1, a2, ….

One proof of this uses Ramsey's Theorem from combinatorics, together with compactness. Given a1, a2, …, write down the partial type Σ(x1, x2, …) consisting of the following statements:

  • ϕ(xj1, …, xjn) ⇔ ϕ(xi1, …, xin) for every n, every formula ϕ(y1, …, yn), and every j1 < ⋯ < jn and i1 < ⋯ < in.
  • ψ(x1, x2, …, xn) for every formula ψ in the EM type of the sequence a1, a2, ….

Any realization of the partial type Σ will be an indiscernible sequence realizing the desired EM type. By compactness, it suffices to check finite satisfiability of this type. More precisely, it suffices to check that for any given ϕ1, …, ϕm, we can satisfy the type Σϕ1, …, ϕm(x1, x2, …) consisting of the following statements.

  • ϕi(xj1, …, xjni) ⇔ ϕi(xi1, …, xini) for 1 ≤ i ≤ m and every j1 < ⋯ < jni and i1 < ⋯ < ini.
  • ψ(x1, x2, …, xn) for every formula ψ in the EM type of the sequence a1, a2, ….

Here, ni denotes the number of free variables occurring in ϕi = ϕi(y1, …, yni). By adding on dummy variables at the end, it is safe to assume that ni doesn't depend on i.

So Σϕ1, …, ϕm basically expresses the desired EM type plus "indiscernibility with respect to the formulas in {ϕ1, …, ϕm}."

Now we "color" each of the n-element subsets of ℕ with one of 2m colors. Specifically, the "color" attached to {i1, …, in} ⊂ ℕ is

{k ≤ m:⊨ ϕk(ai1, …, ain)} ⊂ {1, …, m}.

By Ramsey's Theorem, we can find an infinite homogeneous subset of ℕ. That is, we can find c1 < c2 < ⋯ and a color C such that for any i1, …, in, the color attached to {ci1, …, cin} is C. In particular, if i1 < ⋯ < in and j1 < ⋯ < jn, then the same color is attached to {ci1, …, cin} as to {cj1, …, cjn}. This means that for 1 ≤ k ≤ m, we have

 ⊨ ϕk(aci1, …, acin) ⇔ ϕk(acj1, …, acjn).

So the subsequence

ac1, ac2, …

of the original sequence a1, a2, … satisfies the "indiscernibility with respect to <\math>\{\phi_1, \ldots, \phi_m\}</math>" condition. As a subsequence of a1, a2, …, it also satisfies the EM type of a1, a2, …. Therefore, it satisfies Σϕ1, …, ϕm.

So each type Σϕ1, …, ϕm is satisfiable. But any finite subset of Σ is contained in one of these, so Σ is finite satisfiable. QED.

Another proof, more model-theoretic than combinatorial, uses the machinery of global invariant types. Without developing that machinery (and the machinery of monster models), we describe the proof.

Lemma: Let M be a structure containing a sort I with a total ordering <, such that I has no greatest element. Then in some elementary extension of M, we can find an indiscernible increasing sequence in I.

Proof: Let M′ ≽ M be an |M|+-saturated elementary extension of M. Consider the partial type Σ(x) over M′, where x is a variable in the sort I, consisting of the following formulas:

  • ϕ(x, a) ⇔ ϕ(x, b), whenever a≡Mb and ϕ(x; y) is a formula.
  • x > α for every α ∈ I(M). (Here, I(M) denotes the interpretation of I in M, or equivalently, I ∩ M.)

We claim that Σ(x) is consistent. Note that any element of I(M) will certainly satisfy the first type of statement, by definition of what a≡Mb means. By compactness, it therefore suffices to show that any finite collection of statements of the second type is realized by someone in I(M). That is, we need to show that if α1, …, αn are in I(M), then some element of I(M) is greater than each αi. This is immediate from the fact that I has no greatest element.

Now let p(x) be a complete type over M′ extending the consistent partial type Σ(x). Build a sequence α1, α2, … ∈ M′ recursively by letting αn be any realization of p restricted to M ∪ {α1, …, αn − 1}. We can find a realization of this type in M′ because we assumed M′ to be |M|+-saturated.

We claim that α1, α2, … is not just indiscernible, but even M-indiscernible. To do this, we prove by induction on n that if i1 < ⋯ < in and j1 < ⋯ < jn, then

αi1⋯αin≡Mαj1⋯αjn.

The base case of n = 1 amounts to showing that each αi has the same type over M. But this is clear, since each αi realized p|M (among other things).

For the inductive step, suppose we know the claim for all n < m, and consider n = m. Given i1 < ⋯ < im and j1 < ⋯ < jm, we first find some k such that k > max(im, jm). Then αk realizes p restricted to M ∪ {α1, …, αk}. In particular, αk realizes p ↾ M ∪ {αj1, …, αjm − 1}. So does αjm, so

αk≡Mαj1⋯αjm − 1αjm

or equivalently,

αj1⋯αjm − 1αk≡Mαj1⋯αjm.

Similarly,

αi1⋯αim − 1αk≡Mαi1⋯αim.

So it suffices to show that

αj1⋯αjm − 1αk≡Mαi1⋯αim − 1αk.

By the inductive hypothesis,

αj1⋯αjm − 1≡Mαi1⋯αim − 1.

Now if ϕ(x1, …, xm) is any formula over M, then the formula

ϕ(αi1, …, αim − 1, x) ⇔ ϕ(αj1, …, αjm − 1, x)

is in Σ(x), hence in p(x), and hence in p ↾ M ∪ {α1, …, αk − 1}. In particular, αk satisfies it.

So

ϕ(αi1, …, αim − 1, αk) ⇔ ϕ(αj1, …, αjm − 1, αk)

holds for every formula ϕ(x1, …, xm) over M. This means precisely that

αj1⋯αjm − 1αk≡Mαi1⋯αim − 1αk,

completing the inductive step.

So we conclude that α1, α2, … is M-indiscernible, hence indiscernible. If α1 < α2 < ⋯, then we are done. Otherwise, α1 > α2 > ⋯. Consider the reversed sequence …, α3, α2, α1. This is also indiscernible. By the Lemma at the beginning of this section, we can pass to an even bigger elementary extension and find an indiscernible sequence b1, b2, … realizing the EM type of …, α3, α2, α1, which includes the statement x1 < x2. Then b1, b2, … is our desired increasing indiscernible sequence.

QED.

Now to prove the Theorem on the ability to extract indiscernible sequences, it remains to prove the following:

Lemma: Let M be some model. Let a1, a2, … be a sequence from M. Then we can find an indiscernible sequence b1, b2, … realizing the EM type of the sequence a1, a2, ….

Proof: Consider the expansion of M to a new structure N in which we have added a sort I interpreted as ℕ, as well as the binary order predicate on I and a unary function symbol f from I to M sending i ∈ ℕ to ai. By the Lemma, we can find an elementary extension N′ ≽ N containing an increasing indiscernible sequence α1, α2, … in the sort I. Let M′ be the reduct of N′ to the original language of M. One easily checks that M′ ≽ M. The function f is Ø-definable, so the sequence f(α1), f(α2), … is an (Ø-)indiscernible sequence in N′. It lives in M′, and is indiscernible in the reduct M′ (because there are fewer formulas for which to check indiscernibility). So we have an indiscernible sequence {f(αi)}i ∈ ℕ in M′, an elementary extension of M.

It remains to check that this sequence realizes the EM type of the original sequence a1, a2, …. Let ϕ(x1, …, xn) be a formula in that EM type. Then

N ⊨ ∀x1, …, xn ∈ I : x1 < x2 < ⋯ < xn ⇒ ϕ(f(x1), …, f(xn))

(This is a restatement of what it means for ϕ to be in the EM type.) Since N′ ≽ N, the above also holds in N. Now given i1 < ⋯ < in, note that

N′ ⊨ αi1 < αi2 < ⋯ < αin,

so we must also have

N′ ⊨ ϕ(f(αi1), f(αi2), …, f(αin).

Since ϕ was a formula in the original language, it follows that

M′ ⊨ ϕ(f(αi1), …, f(αin).

Consequently, ϕ is in the EM type of {f(αi)}i ∈ ℕ. This completes the proof. QED.
