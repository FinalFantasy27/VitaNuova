Recall that in ACF, Morley rank, Krull dimension, and Lascar rank all agree, and dimension is definable.

Lemma. Let V be a Zariski closed subset of ℙn, with dim V ≤ n − 2, with V definable over some set C. Let p be a generic point in ℙn, generic over C. Then p ∉ V (obviously). Let P′ be the projective space of lines in ℙn passing through p. Then there is a natural projection map π : ℙn ∖ p ↠ P′. The image π(V) is a Zariski closed subset of P′ (because V was proper/complete).

Then,

(a)
If W is an irreducible component of V, and q is a generic point on W (generic over p and C), then the projection V → π(V) is injective at q, i.e., there is no q′ ∈ V with q′ ≠ q but π(q′) = π(q).
(b)
π(V) is irreducible if and only if V is.

Proof.

(a)

We need to show that if we fix V, choose p generic in ℙn, and then choose q generic in W, that the line L through p and q intersects V only at q. By symmetry of forking, we can instead choose q first, and then choose p generic over C and q. If we make the choices in this order, then L is a generic line through q. In particular, U([L]/Cq) = n − 1, where [L] denotes the code for the set L. Suppose L intersects V at some other point q′. Then [L] ∈ dcl(qq′), so n − 1 > U(V) ≥ U(q′/Cq) ≥ U([L]/Cq) = n − 1.

(b)

If V is irreducible, then π(V) is irreducible, on general grounds. Conversely, suppose V is not irreducible but π(V) is irreducible. Let C′ be the union of C and the codes for the irreducible components of V. Then C′ and C have the same algebraic closure, U(p/C′) = U(p/C) and p is still generic over C′. Replacing C with C′, we may assume that each irreducible component of V is C-definable. Write V as the union of its irreducible components W1 ∪ W2 ∪ ⋯ ∪ Wn, with dim W1 ≤ dim W2 ≤ ⋯ ≤ dim Wn = dim V. Let (q1, …, qn) be a generic point in W1 × ⋯ × Wn, generic over Cp. By part (a), the map V → π(V) is injective at each qi, so qi and π(qi) are interdefinable over Cp, for each i. Therefore U(π(qi)/Cp) = U(qi/Cp) for each i.

Consequently, dim V ≥ dim π(V) ≥ dim π(Wn) ≥ U(π(qn)/Cp) = U(qn/Cp) = dim Wn = dim V, so π(Wn) and π(V) have the same dimension. But π(Wn) is irreducible because Wn is, and π(V) is irreducible by assumption. Therefore π(Wn) = π(V).

Meanwhile, W1 ≠ Wn, so W1 ∩ Wn is strictly smaller than W1. Consequently dim π(W1 ∩ Wn) ≤ dim W1 ∩ Wn < dim W1. Since U(π(q1)/Cp) = U(q1/Cp) = dim W1 > dim π(W1 ∩ Wn), we have π(q1) ∉ π(W1 ∩ Wn). However π(q1) ∈ π(V) = π(Wn), so π(q1) = π(r) for some r ∈ Wn ∖ W1, contradicting injectivity of π at q1.

QED

Theorem. Let ϕ(x; y) be a formula such that for every b, the set ϕ(𝕌; b) is a Zariski closed subset of ℙn. Then the set of b such that ϕ(𝕌; b) is irreducible is definable.

Proof. We proceed by induction on n, the case n = 1 being extremely easy. Let Vb = ϕ(𝕌; b). Since Morley rank is definable in ACF, the set of b such that dim Vb < n − 1 is definable, as is the set of b such that dim Vb = n. Breaking into cases, we may assume one of the following:

  • dim Vb = n whenever Vb is non-empty.
  • dim Vb = n − 1 whenever Vb is non-empty.
  • dim Vb < n − 1 whenever Vb is non-empty.

In the first case, Vb is always irreducible (whenever it is non-empty). In the second case, Vb is irreducible if and only if it is non-empty and the zero set of some irreducible homogeneous polynomial. The set of irreducible homogeneous polynomials of degree d is definable, for each d, so the set of b such that Vb is irreducible is ind-definable. So is the set of b such that Vb is not irreducible. (For each m, the family ℱn of all Zariski closed subsets of ℙn which are cut out by at most m polynomials of degree at most m is a uniformly definable family. The family ℛn of all Zariski closed sets of the form W1 ∪ W2 with W1 ⊈ W2 ⊈ W1 and Wi ∈ ℱn is also a uniformly definable family. But $\bigcup\limits_{n = 1}^{\infty}\mathcal{R}_{n}$ is the collection of all reducible Zariski closed sets.) Consequently, the set of b such that Vb is reducible is definable.

In the third case, proceed as follows. For p ∈ ℙn, let πp be the projection to ℙn − 1 with center at p. If p is generic over b, then πp(Vb) is irreducible if and only if Vb is. By induction, the set of (p, b) such that πp(Vb) is irreducible is definable. By definability of types in stable theories, the set of b such that πp(Vb) is irreducible for generic p is definable. But this is the set of b such that Vb is irreducible. QED

It is easy to prove by induction on dim D that if D is a definable (= constructible) subset of ℙn, then D is a finite union of sets of the form V ∩ U where V is an irreducible Zariski closed set and U is a Zariski open set intersecting V. The Zariski closure of V ∩ U is exactly V. (If it were W, then V ∩ U ⊂ W, so V ⊂ W ∪ (V ∖ U). Since W ⊂ V and V ∖ U ⊂ V, either W = V or V ∖ U = V. In the first case, we are done; in the second U does not intersect V.) Writing $D = \bigcup\limits_{i = 1}^{m}V_{i} \cap U_{i},$ it follows that $\overset{¯}{D} = \bigcup\limits_{i = 1}^{m}V_{i}.$

Let ℱm be the definable family of all constructible sets of the form $\bigcup\limits_{i = 1}^{k}V_{i} \cap U_{i}$ where k ≤ m, and each Vi and Uic is cut out by at most m polynomials of degree at most m, and Vi is irreducible. By the Theorem, ℱm is a uniformly definable family. By the previous paragraph, the map assigning to an element of ℱm its Zariski closure is also definable.

Now every constructible set is in ℱm for sufficiently large m. So if ϕ(x; y) is a formula, then compactness ensures that there is some m such that for every b, ϕ(𝕌; b) ∈ ℱm. It follows that the Zariski closure of ϕ(𝕌; b) is uniformly definable from b. In other words,

Corollary. Zariski closure is definable in families. That is, if ϕ(𝕌; b) ⊂ ℙn for every b, then there is some formula ψ(x; y) such that ψ(𝕌; b) is the Zariski closure of ϕ(𝕌; b) for every b.

From this, we conclude that:

  • The property of being Zariski closed (in ℙn) is definable in families.
  • The property of being Zariski closed in 𝔸n is definable in families. (Use the embedding of 𝔸n into ℙn.)
  • The property of being Zariski closed and irreducible in 𝔸n is definable in families. (A closed subset of 𝔸n is irreducible if and only if its Zariski closure in ℙn is irreducible.) ::: ::: :::
