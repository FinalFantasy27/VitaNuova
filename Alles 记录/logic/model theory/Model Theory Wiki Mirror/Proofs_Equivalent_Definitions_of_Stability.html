Equivalent definitions of stability[]

Fix some complete theory T. Let 𝕌 be a monster model for T.

By default, "types" will be complete types. Types will be finitary types (n-types for some n), unless stated otherwise.

A formula ϕ(x; y) has the order property if there exist ai, bi for i < ω such that  ⊨ ϕ(ai; bj) if and only if i < j. Equivalently, by compactness, for every n, there exist ai, bi for i < n such that  ⊨ ϕ(ai; bj) if and only if i < j.

Let κ ≥ |T| be a cardinal. We say that T is κ-stable if there are at most κ complete types over any set of size at most κ. That is, if for every set A ⊂ 𝕌 such that |A| ≤ κ, we have |Sn(A)| ≤ κ for every n.

If p(x) is a type over a set A, we say that p(x) is definable (over A) if for every formula ϕ(x; y), there is some A-formula ψ(y) such that for each a ∈ A, ϕ(x; a) ∈ p(x) ⇔  ⊨ ψ(a)

Theorem. The following are equivalent for T a complete theory:

  • Every type over every set is definable
  • T is κ-stable for some κ
  • T is κ-stable for every infinite cardinal κ ≥ |T| such that κ|T| = κ.
  • No formula has the order property.

We say that T is stable if it satisfies these equivalent conditions.

There are a number of additional conditions which are also equivalent to stability:

  • Every 1-type over every set is definable.
  • Every type over every model is definable.
  • Every 1-type over every model is definable.
  • Every indiscernible sequence is totally indiscernible.
  • No formula ϕ(x; y) with x a singleton has the order property.
  • There is some κ ≥ |T| such that for every set A with |A| ≤ κ, we have |S1(A)| ≤ κ, where S1(A) is the space of 1-types over A. In other words, we have stability on the level of 1-types.
  • For every κ ≥ |T| with κ|T| = κ, if |A| ≤ κ then |S1(A)| ≤ κ.
  • For every formula ϕ(x; y), there are only countably many ϕ-types over any countable set A.
  • For every formula ϕ(x; y) and set A, the space Sϕ(A) of ϕ-types over A contains no perfect set, or equivalently, has Cantor-Bendixson rank less than ∞.
  • No formula has the independence property and no formula has the strict order property (T is both NIP and NSOP). The proof of this is purely combinatorial and has nothing to do with the rest of this article; it can be found here.
  • T is both NIP and simple

Below, we will prove all or most of these equivalences.

[From definability of types to κ-stability]{#From_definability_of_types_to_‘"UNIQ--postMath-00000043-QINU"’-stability .mw-headline}[]

Lemma 1. Suppose every 1-type over a model is definable. Suppose κ|T| = κ ≥ |T| and A ⊂ 𝕌 has |A| ≤ κ. Then |S1(A)| ≤ κ.

Proof. By Löwenheim-Skolem, we can find a model M ⊇ A with |M| ≤ |A| + |T| ≤ κ. The restriction map S1(M) → S1(A) is surjective, so it suffices to show |S1(M)| ≤ κ. By assumption, every 1-type over M is definable. A definition consists of a map which assigns to each formula ϕ(x; y) an M-formula ψ(y). There are |T| formulas and κ + |T| = κ formulas over M, so there are at most κ|T| = κ possible definitions. Consequently, there are at most κ-many 1-types over M. QED

Lemma 2. Let κ ≥ |T| be a cardinal. Suppose that for every set A ⊂ 𝕌 with |A| ≤ κ we have |S1(A)| ≤ κ. Then T is κ-stable, i.e., for every set A ⊂ 𝕌 with |A| ≤ κ and every n, |Sn(A)| ≤ κ.

Proof. Let A be a set of size at most κ. We need to show that there are at most κ-many n-types over A.

The rough idea here is that to specify tp(a1a2⋯an/A), we need only specify tp(an/A), tp(an − 1/Aan), tp(an − 2/Aanan − 1), etc. At each step there are at most κ choices, so all together there are κn = κ possibilities.

More rigorously, we proceed by induction on n. The case n = 1 is given. Suppose we know |Sn − 1(A)| ≤ κ. For each (n − 1)-type over A, choose some realization, and let B be the set of all these realizations. For each b in B, there are at most κ-many 1-types over Ab, by the assumption. For each 1-type over Ab choose some representative realization and let Cb be the set of representatives. So |Cb| ≤ κ. Now if D = {(c, b) : b ∈ B, c ∈ Cb} the |D| ≤ κ × κ = κ, and so it suffices to show that every n-type over A is realized in D. Let p be an n-type over A, and let (c′, b′) be a tuple realizing p, where c′ is a singleton and b′ is an (n − 1)-tuple. As B was a complete set of representatives of the (n − 1)-types over A, there is some b ∈ B such that b≡Ab′. Let σ ∈ Aut(𝕌/A) send b′ to b. As Cb is a complete set of representatives of the 1-types over Ab, we can find some c ∈ Cb such that c≡Abσ(c′), or equivalently, (c, b)≡A(σ(c′), b). But then (c, b)≡A(σ(c′), b) = σ((c′, b′))≡A(c′, b′), where the second ≡A follows because σ is an automorphism over A. So p = tp(c′b′/A) = tp(cb/A), and p is realized by someone in D. This completes the inductive proof. QED

From stability to the order property[]

If (S,  < ) is a totally ordered set, by a Dedekind cut of S we will mean a downward-closed subset T ⊂ S, so if s, s′ ∈ S with s < s′, then s′ ∈ T ⇒ s ∈ T.

Lemma 3. Let κ be a cardinal. Then there exists a totally ordered set (S,  < ) of size at most κ with more than κ Dedekind cuts.

Proof. Let λ be the smallest cardinal such that 2λ > κ. As 2κ > κ, we have λ ≤ κ. Let S be 2 < λ, the set of all strings of 0’s and 1’s of length less than λ. We will produce a Dedekind cut on S for each element of 2λ = {0, 1}λ, the set of strings of 0’s and 1’s of length λ.

Note that $\left| 2^{< \lambda} \right| \leq \sum\limits_{\alpha < \lambda}2^{|\alpha|} \leq \sum\limits_{\alpha < \lambda}\kappa \leq \lambda \times \kappa = \kappa.$ Let T be the set of all functions from λ to {0, *,1}, ordered lexicographically with 0 <  *  < 1. Embed S into T by taking a string s and appending λ copies of * to the end. We claim that the induced ordering on S has more than κ-many Dedekind cuts. In fact, each element of {0, 1}λ (a set of size 2λ > κ) induces a distinct Dedekind cut on S. To see this, let w < w′ be two distinct strings from {0, 1}λ ⊂ T. Then w and w′ differ at some place, and we have w = v0u w′ = v1u′ for some v, u, u′. Now w < v < w′ where we are viewing v ∈ S as an element of T via the embedding S ↪ T. So w and w′ induce different Dedekind cuts on S. QED

Lemma 4. Suppose T is κ-stable for some T. Then no formula ϕ(x; y) has the order property.

Proof. By Lemma 3, we can find a totally ordered set (S,  < ) such that |S| ≤ κ but S has more than κ Dedekind cuts.

Suppose ϕ(x; y) has the order property. So there exist ai, bi for i < ω such that ϕ(ai; bj) holds if and only if i < j. Let ⟨csds⟩s ∈ S be an S-indexed indiscernible sequence extracted from ⟨aibi⟩i < ω. So ϕ(cs; ds′) holds if and only if s < s′.

Now if s1 < ⋯ < sn < t1 < ⋯ < tm is an ascending sequence in S, then we can find some b ∈ 𝕌 such that ϕ(asi; b) holds for 1 ≤ i ≤ n and ¬ϕ(ati; b) holds for 1 ≤ i ≤ m. Namely, we take b to be bt1. By compactness, it follows that for each Dedekind cut D of S, we can find some bD ∈ 𝕌 such that ϕ(as; bD) holds for s ∈ D and doesn’t hold for s ∉ D.

Let A be the set {as : s ∈ S}. Then |A| ≤ |S| ≤ κ. On the other hand, if D ≠ D′, then bD≢AbD′, because if s ∈ D ∖ D′, then ϕ(as; bD) holds and ϕ(as; bD′) does not hold. There are more than κ-many D’s, so there are more than κ-many types over A, contradicting κ-stability. QED

From the order property to definability[]

The missing link in the proof is that no formula having the order property implies the definability of types over models. This is where things get tricky.

Step 1: Total indiscernibility[]

Lemma 5. Fix n < ω. Suppose that no formula ϕ(x; y) has the order property with x an n-tuple of variables. Then every A-indiscernible sequence of n-tuples is totally A-indiscernible.

Proof. Let ⟨ai⟩i ∈ I be some A-indiscernible sequence. Whether or not ei is totally A-indiscernible is part of the Ehrenfeucht-Mostowski type over A of the sequence, so we can replace ai with another A-indiscernible sequence having the same EM type. Replacing ai with a ℚ-indexed A-indiscernible sequence extracted from ⟨ai⟩i ∈ I, we may assume that I is ℚ (with the usual ordering).

Let x1 < ⋯ < xn ∈ ℚ, and 1 ≤ i < n. We claim that ax1⋯axn≡Aax1⋯axi − 1axi + 1axiaxi + 2⋯axn, i.e., that we can swap two consecutive elements in the sequence (ax1, …, axn) without changing the type of the sequence.

If not, then axiaxi + 1≢Baxi + 1axi where B = Aax1⋯axi − 1axi + 2⋯axn. Thus there is some B-formula ϕ(x; y) (with x and y both n-tuples) such that ϕ(axi, axi + 1) holds and ϕ(axi + 1, axi) does not. Now ⟨ax : xi − 1 < x < xi + 2⟩ is a B-indiscernible sequence. Therefore, if y, z ∈ (xi − 1, xi + 2), then ϕ(ay, az) holds if y < z and doesn’t hold if z > y. By density of ℚ, we can choose yi, zi for i < ω such that xi − 1 < z1 < y1 < z2 < y2 < z3 < y3 < ⋯ < xi + 2. Now ϕ(ayi; azj) holds if and only if i < j. Writing ϕ(x; y) as ψ(x; y, b) for b some tuple of parameters from B, we have  ⊨ ψ(ayi; azjb) ⇔ i < j Consequently ψ(x;  − ) is a formula with the order property and x is a tuple of length n, contradicting the hypothesis.

Therefore the claim is proven, and we can swap any two consecutive elements of a1a2⋯an without changing the type over A. This property remains true after performing the swap. Since every permutation can be written as a product of such swaps, it follows that every permutation of a1a2⋯an has the same type over A. So the sequence is totally A-indiscernible. QED

Lemma 6. Let ϕ(x; y) be a formula without the order property, and n be the length of the tuple x. There is an integer N, depending only on ϕ(x; y), such that for every totally ⌀-indiscernible sequence {ai : i ∈ I} and every b, one of the two sets {i ∈ I : 𝕌 ⊨ ϕ(ai; b)} {i ∈ I : 𝕌 ⊨ ¬ϕ(ai; b)} has size at most N.

Proof. If not, then for each N we can find a totally indiscernible sequence {ai : i ∈ I} and a b such that ϕ(ai; b) holds for more than N values of i, and fails to hold for more than N values of i. Any permutation or infinite subset of a totally indiscernible sequence is still totally indiscernible. So we can pass to a countable subsequence and rearrange the sequence to be indexed by ℤ. We can reorder the terms so that ϕ(ai; b) holds for i =  − 1, …,  − (N + 1) and doesn’t hold for i = 0, …, N.

Now since we can do this for each N, by compactness we can find a totally indiscernible sequence {ai : i ∈ ℤ} and a b such that ϕ(ai; b) holds if and only if i < 0.

By indiscernibility, for each j we can find a σj ∈ Aut(𝕌) such that σj(ai) = ai + j. Then  ⊨ ϕ(ai; σj(b)) ⇔  ⊨ ϕ(σj − 1(ai); b) ⇔  ⊨ ϕ(ai − j; b) ⇔ i − j < 0 So ϕ(x; y) has the order property, a contradiction. QED

Step 2: Morley Sequences[]

Suppose that no formula ϕ(x; y) with x of length n has the order property.

Let p(x) be an n-type over 𝕌 which is Aut(𝕌/A)-invariant for some small set A. A Morley sequence for p is a sequence ⟨ai⟩i ∈ I such that for each i, a realizes p restricted to A ∪ {aj : j < i}. Observe that any subsequence of a Morley sequence is a Morley sequence.

Lemma. Any two Morley sequences of length m have the same type over A.

Proof. By induction on m. For m = 1, we have a1, b1 both realizing p|A. Then clearly a1≡Ab1. Suppose all Morley sequences of length m − 1 have the same type over A. Let a1, …, am and b1, …, bm be two Morley sequences of length M. By the inductive hypothesis, we can find σ ∈ Aut(𝕌/A) such that σ(bi) = ai for i < m. Now bm realizes p|Ab1b2⋯bm − 1, so σ(bm) realizes σ(p)|Aa1⋯am − 1. Since p is Aut(𝕌/A)-invariant, σ(bm) really realizes p|Aa1⋯am − 1, which is the same type that am realizes. Thus am≡Aa1⋯am − 1σ(bm) or equivalently a1a2⋯am≡Aa1a2⋯am − 1σ(bm) = σ(b1b2⋯bm). Since σ(b1b2⋯bm)≡Ab1b2⋯bm we conclude that a1⋯am and b1⋯bm have the same type over A. QED

Lemma. Any Morley sequence is A-indiscernible, hence totally A-indiscernible by Lemma 5.

Proof. Let ⟨ai⟩i ∈ I be a Morley sequence. It suffices to show that any two finite subsequences of the same length have the same type over A. Since they are both Morley sequences, this follows from the previous lemma. QED

Lemma 7. Let ⟨ai⟩i ∈ I be an infinite Morley sequence for p. Let q(x) be the set of all formulas ϕ(x; b) (with b from 𝕌) such that 𝕌 ⊨ ϕ(ai; b) for all but at most finitely many values of i. Then q(x) is a consistent global type, which is definable (over some small set, not necessarily A). The type q is called the average type of ⟨ai⟩i ∈ I.

Proof. By the previous lemma, ⟨ai⟩i ∈ I is totally indiscernible over A, hence over ⌀. Therefore, by Lemma 6, we know that for every 𝕌-formula ϕ(x; b), either ϕ(ai; b) holds for almost all i, or ¬ϕ(ai; b) holds for almost all i.

First we check that q(x) is consistent. If not, then there exist 𝕌-formulas ϕ1(x), …, ϕn(x) in q(x) which are jointly inconsistent. But for each 1 ≤ j ≤ n, at most finitely many ai fail to satisfy ϕj(x). Therefore, at most finitely many ai fail to satisfy $\bigwedge\limits_{j = 1}^{n}\phi_{j}(x)$. Since I is infinite, at least one i satisfies $\bigwedge\limits_{j = 1}^{n}\phi_{j}(x)$. So q(x) is consistent.

Lemma 6 ensures that q(x) is complete, i.e., that ϕ(x) or ¬ϕ(x) is in q(x) for every 𝕌-formula ϕ(x).

Definability comes from the number N in Lemma 6. Specifically, for each formula ϕ(x; y) we need to show that {b ∈ 𝕌 : ϕ(x; b) ∈ q(x)} is definable. Lemma 6 gives us a number N = Nϕ depending on ϕ (but not on b) such that for every b, either

  • 𝕌 ⊨ ϕ(ai; b) for all but at most Nϕ values of i, in which case ϕ(x; b) ∈ q(x); OR
  • 𝕌 ⊨ ϕ(ai; b) for no more than Nϕ values of i, in which case ¬ϕ(x; b) ∈ q(x).

Consequently, we can determine whether ϕ(x; b) belongs in q(x) by checking ϕ(ai; b) for 2Nϕ + 1 values of i, and doing a majority vote. If i1, …, i2Nϕ + 1 are 2Nϕ + 1 distinct values of i, then we get $\phi(x;b) \in q(x)\;\Longleftrightarrow\;\mathbb{U} \vDash \bigvee\limits_{S \subset 2N_{\phi} + 1,\ |S| \geq N_{\phi} + 1}\bigwedge\limits_{i \in S}\phi(a_{i};b),$ and so $\bigvee\limits_{S \subset 2N_{\phi} + 1,\ |S| \geq N_{\phi} + 1}\bigwedge\limits_{i \in S}\phi(a_{i};x)$ is the ϕ-definition of q. QED

[Step 3: p is the average type of its Morley sequences]{#Step_3:_‘"UNIQ--postMath-000001C6-QINU"’_is_the_average_type_of_its_Morley_sequences .mw-headline}[]

Continue to assume that no formula ϕ(x; y) with x of length n has the order property, and that p(x) is a global n-type which is A-invariant.

Lemma. If I′ is a Morley sequence for p and I is an infinite subsequence, then I (which is also a Morley sequence) has the same average type as I′.

Proof. Let q′ and q be the average types of I′ and I. If ϕ(x) ∈ q′(x), then ϕ(a) holds for all but finitely many a in I′. In particular, ϕ(a) also holds for all but finitely many a in I, so ϕ(x) ∈ q(x). Thus q′ ⊂ q. Since q′ and q are complete types, they must be equal. QED

Lemma. If I and I′ are two infinite Morley sequences for p, then they have the same average type.

Proof. Recursively build a sequence a1, a2, … by letting ai realize p restricted to A ∪ I ∪ I′ ∪ {aj : j < i}. Let J be the sequence of all the ai’s. Then the concatenations IJ and I′J are Morley sequences. By the previous lemma, Av(I) = Av(IJ) = Av(J) = Av(I′J) = Av(I′), where Av(I) denotes the average type of a Morley sequence I. QED

So there is a unique global type q(x) which is the average type of any Morley sequence of p(x).

Lemma 8. q = p

Proof. Let ϕ(x; b) be a formula in p(x). We will show that ϕ(x; b) ∈ q(x). This suffices because p(x) and q(x) are complete types (over the monster). Let a1, a2, … be a sequence built up recursively by letting ai realize the restriction of p to Ab ∪ {aj : j < i}. Then a1, a2, … is a Morley sequence for p. The formula ϕ(x; b) is in the restriction of p to Ab, so ai satisfies it for every i. That is, ϕ(ai; b) holds for every i. It follows that ϕ(x; b) is in the average type of a1, a2, …, but this average type is q(x). QED

Combining this with Lemma 7, we get

Lemma 9. Suppose that no formula ϕ(x; y) with x of length n has the order property. Let p(x) be a global n-type. Suppose that p(x) is A-invariant for some small set A. Then p is A-definable, i.e., for every formula ϕ(x; y) the set {b ∈ 𝕌 : ϕ(x; b) ∈ p(x)} is an A-definable subset of 𝕌.

Proof. Lemma 8, p is the average type of any Morley sequence. By Lemma 7, it follows that p is definable over some set. So for each formula ϕ(x; y), the set {b ∈ 𝕌 : ϕ(x; b) ∈ p(x)} is definable. But it is also Aut(𝕌/A)-invariant, so it must be A-definable. QED

Finally, we can deduce the definability of types (over models) from the absence of formulas with the order property:

Lemma 10. Suppose that no formula ϕ(x; y) with x of length n has the order property. Then every n-type p(x) over a model M is M-definable.

Proof. Let Σ(x) be the global partial type which consists of all formulas of the form ϕ(x; b) ↔︎ ϕ(x; c) for b≡Mc. Note that any element of M satisfies Σ(x). If p0(x) is a finite subset of p(x), then p0(x) is satisfiable in M, hence so is p0(x) ∪ Σ(x). It follows that p(x) ∪ Σ(x) is consistent. Let p′(x) be a global type extending p(x) ∪ Σ(x). Then whenever b≡Mc, we have ϕ(x; b) ∈ p′(x) ⇔ ϕ(x; c) ∈ p′(x). Therefore p′(x) is Aut(𝕌/M)-invariant, hence M-definable by Lemma 9. But p(x) is the restriction of p′(x) from 𝕌 to M, so it is also M-definable. QED

The first few definitions of stability[]

We have now reached the following:

Theorem. The following are equivalent:

(a)
Every type over a model is definable.
(a')
Every 1-type over a model is definable.
(b)
T is κ-stable for all κ ≥ |T| with κ|T| = κ.
(b')
same as (c) but for 1-types rather than n-types
(c)
T is κ-stable for some κ.
(c')
same as (d) but for 1-types rather than n-types
(d)
No formula ϕ(x; y) has the order property.
(d’)
No formula ϕ(x; y) with x a singleton has the order property.

Proof. Clearly (c) implies (c'), and Lemma 2 gives the converse. Similarly, (b) and (b') are equivalent.

We have the following sequence of implications: (c) ⇒ (d) ⇒ (d′) ⇒ (a′) ⇒ (b′) ⇔ (b) (c) ⇒ (d) ⇒ (a) ⇒ (a′) ⇒ (b′) ⇔ (b) From left to right:

  • (c)  ⇒  (d) is by Lemma 4.
  • (d)  ⇒  (d') is obvious.
  • (d)  ⇒  (a) is by Lemma 10.
  • (d')  ⇒  (a') is by Lemma 10.
  • (a)  ⇒  (a') is obvious.
  • (a')  ⇒  (b') is by Lemma 1.

It remains to show that (b)  ⇒  (c). Suppose T is κ-stable for every κ ≥ |T| such that κ|T| = κ. Let κ = 2|T|. Then κ|T| = 2|T|2 = 2|T| = κ, so T is κ-stable. In particular, T is κ-stable for at least one κ. QED

We say that T is stable if it satisfies one of these equivalent conditions.

Additional equivalent conditions[]

At this point, one easy equivalence is:

Theorem. T is stable if and only if every indiscernible sequence is totally indiscernible.

Proof. If T is stable, then for every n, no formula ϕ(x; y) with x of length n has the order property. By Lemma 5, every indiscernible sequence is totally indiscernible. Conversely, suppose that T is not stable. Then some formula ϕ(x; y) has the order property. Let ai, bi for i < ω witness this, so that ϕ(ai; bj) holds if and only if i < j. Let ⟨cidi⟩i < ω be an indiscernible sequence extracted from ⟨aibi⟩i < ω. Then ϕ(ci; dj) holds if and only if i < j, because this was expressed in the Ehrenfeucht-Mostowski type of ⟨aibi⟩i < ω. But now ϕ(c2; d3) holds and ϕ(c2; d1) does not, indicating that (c2d2, c3d3) and (c2d2, c1d1) do not have the same type, so the sequence ⟨cidi⟩i < ω is not totally indiscernible, in spite of being indiscernible. QED

Lemma. Let T be a countable stable theory, and ϕ(x; y) be a formula. Then over any countable set A, there are at most countably many ϕ-types.

Proof. Since T is countable, by Löwenheim-Skolem we can find a countable model M containing A. Then Sϕ(M) surjects onto Sϕ(A), so it suffices to show that Sϕ(M) is countable. If p ∈ Sϕ(M), we can extend p to a complete type p′ ∈ S(M). The type p′ is M-definable, by stability. Let ψ(y; m) be the ϕ-definition of p′, with m a tuple from M. Then for b ∈ M, ϕ(x; b) ∈ p(x) ⇔ ϕ(x; b) ∈ p′(x) ⇔  ⊨ ψ(b; m). The ϕ-type p(x) is therefore determined by the formula ψ(y; m). Since T and M are countable, there are only countably many possibilities for ψ(y; m). Hence there are only countably many possibilities for p. QED

Observe that any reduct of a stable theory is still stable. One way to see this is to observe that there are fewer formulas to check against the order property. Alternatively, one can note that there are fewer types in a reduct than in the original structure.

Theorem. T is stable if and only if the following holds: for every formula ϕ(x; y) and every countable set, A, the space of ϕ-types Sϕ(A) is countable.

Proof. Suppose T is stable. Then we can find a countable reduct T′ of T in which ϕ(x; y) is still definable. The space of ϕ-types of A is the same after passing to this reduct, so it is countable.

Conversely, suppose that T is unstable. Let ϕ(x; y) be a formula having the order property, witnessed by ai, bi. Extract an indiscernible sequence ⟨cidi⟩i ∈ ℚ of length ℚ from a1b1, a2b2, …. So ϕ(cx; dy) holds if and only if x ≤ y. By compactness, for each real number r ∈ ℝ, we can find a dr such that x < r ⇔  ⊨ ϕ(cx; dr) Then each dr has a distinct ϕ-type over the countable set {cx : x ∈ ℚ}, so there are uncountably many ϕ-types over this set. QED

[Cantor-Bendixson Rank in the space of ϕ-types]{#Cantor-Bendixson_Rank_in_the_space_of_‘"UNIQ--postMath-000002A5-QINU"’-types .mw-headline}[]

Recall that if B is a boolean algebra, and S(B) is the stone space of ultrafilters on B, then the following conditions are equivalent:

  • S(B) has Cantor-Bendixson rank ∞
  • S(B) contains a perfect set
  • There exist bw ∈ B for w ∈ {0, 1} < ω such that bw ≠ 0 for each w and such that bw0 ∩ bw1 = 0 and bw0 ∪ bw1 = bw for each w.

The first two conditions are equivalent essentially by definition of Cantor-Bendixson rank. If a perfect set P ⊂ S(B) exists, then we can inductively choose b, b0, b1, b00, … as follows:

  • Take b to be any clopen set in S(B) intersecting P.
  • Given bw intersecting P, the set P ∩ bw is still perfect, and hence contains at least two points x0, x1. If U is a clopen neighborhood containing x0 and not x1, then let bw0 = bw ∩ U and bw1 = bw ∖ U.

Then each bw intersects P, hence is non-empty.

Conversely, given bw ≠ 0 such that bw0 ∩ bw1 = 0 and bw0 ∪ bw1 = bw, the Cantor-Bendixson rank must be ∞. If not, then for each w, we can find a w′ ∈ {w0, w1} such that bw′ has lower CB rank than bw, or has the same rank and lower degree. In this way, we obtain a sequence bw, bw′, bw″, … with descending rank ⋅ ω + degree. But there are no infinite descending sequences of ordinal numbers.

By abuse of terminology, let us say that B is "unstable" if one of the above conditions holds, and "stable" otherwise. From the third condition, one sees that if B is stable, so is any subalgebra B′. Conversely, if every countable subalgebra B′ ⊂ B is stable, then B is stable.

Now suppose that T is a stable theory, 𝕌 is a monster model of T, and ϕ(x; y) is a formula. Let B be the boolean algebra generated by ϕ-formulas over 𝕌, so Sϕ(𝕌) = S(B). Then B is stable. If not, then some countable subalgebra B0 ⊂ B is unstable. Then B0 consists of boolean combinations of ϕ-formulas over some countable set of parameters A. If B1 is the boolean algebra generated by all ϕ-formulas over A, then B0 ⊂ B1 ⊂ B, and B1 is unstable. That is, S(B1) = Sϕ(A) contains a perfect set. But then Sϕ(A) is uncountable (by the Cantor-Bendixson theorem), contradicting one of our criteria for stability in the previous section.

Since B is stable, we see that Sϕ(𝕌) has Cantor-Bendixson rank less than ∞, or equivalently, contains no perfect set. The same holds for Sϕ(A) for any subset A ⊂ 𝕌.

Conversely, if T is a theory such that Sϕ(A) contains no perfect set for any A ⊂ 𝕌 and formula ϕ(x; y), then T is stable. Indeed, since Sϕ(A) is a Polish space for countable A, Sϕ(A) must be countable by the Cantor-Bendixson theorem, and therefore T is stable. So we have proven:

Theorem. A theory T is stable if and only if Sϕ(A) contains no perfect set, for every set A and formula ϕ(x; y).

Definability of types over arbitrary sets[]

Lemma. Assume T is stable. Let A be a small subset of 𝕌, p(x) be a complete type over A, and ϕ(x; y) be a formula. Then there exists a global ϕ-type q(x) ∈ Sϕ(𝕌) which is consistent with p(x) and which has finite orbit under Aut(𝕌/A).

Proof. Let X ⊂ Sϕ(𝕌) be the closed subset of Sϕ(𝕌) consisting of the global ϕ-types which are consistent with p. By stability, Sϕ(𝕌) contains no perfect set; therefore neither does X. The boolean space X therefore has Cantor-Bendixson rank less than ∞. Let q be any point in X with maximal Cantor-Bendixson rank. If σ ∈ Aut(𝕌/A), then by symmetry, σ(q) is another point of maximal Cantor-Bendixson rank. But there are only finitely many points of maximal Cantor-Bendixson rank in X. So q has finitely many conjugates. QED

Theorem. A theory T is stable if and only if every type over every set is definable.

Proof. Clearly if every type over every set is definable, then every type over a model is definable, so T is stable.

Conversely, suppose T is stable, and A is a set. Let p(x) be a complete type over A. Let ϕ(x; y) be a formula. Let q(x) be a complete ϕ-type over 𝕌 extending p, with finitely many conjugates over A. Then q(x) is definable, so there is some 𝕌-definable set D such that for all a, ϕ(x; a) ∈ q(x) ⇔ a ∈ D. Now if a ∈ A and σ ∈ Aut(𝕌/A), then ϕ(x; a) ∈ p(x) ⇔ ϕ(x; σ(a)) ∈ q(x) ⇔ ϕ(x; a) ∈ σ(q) ⇔ a ∈ σ(D). Since q has finitely many conjugates over A, so does D. If D′ is the intersection of all the conjugates of D, then D′ is a definable set, which is A-invariant, hence A-definable. And for a ∈ A, ϕ(x; a) ∈ p(x) ⇔ a ∈ D′. So p(x) has a ϕ-definition. As ϕ was arbitrary, p(x) is A-definable. QED

From this, we get the stable embeddedness of all definable sets (what Poizat calls the "Parameter Separation Theorem").

Theorem. Assume T is stable. Let D be a definable set (over some parameters). Then D is stably embedded, i.e., if D′ is a definable subset of Dn for some n, then D′ ∩ Dn = D″ ∩ Dn for some D″ definable over parameters from D.

Proof. Let D′ be cut out by a formula ϕ(b; y), with b some element from the monster. The type of b over D(𝕌) is definable over D(𝕌). In particular, for the formula ϕ(x; y), there is some D(𝕌)-formula ψ(z) such that for every a from D(𝕌), we have a ∈ D′ ⇔  ⊨ ϕ(b; a) ⇔ ϕ(x; a) ∈ tp(b/D(𝕌)) ⇔  ⊨ ψ(a). So if D″ is the set cut out by ψ(z), then a ∈ D′ ⇔ a ∈ D″. This holds for a ∈ Dn, so D′ ∩ Dn = D″ ∩ Dn, where D″ = ψ(𝕌) is definable with parameters from D(𝕌). QED ::: ::: :::
