A sequence of formulas ϕi(x; ai) is said to be k-inconsistent if for every {i1, …, ik} of size k, $\bigwedge\limits_{j = 1}^{k}\phi_{i_{j}}(x;a_{i_{j}})$ is inconsistent. That is, a sequence of formulas is k-inconsistent if any k of the formulas in the sequence is jointly inconsistent. For example, 2-inconsistency is equivalent to pairwise inconsistency.

Typically, k-inconsistency is only considered when the ϕi(x; y) are all the same formula.

This notion is rigged to behave very well with respect to indiscernible sequences. Specifically:

  • If b1, b2, … is an indiscernible sequence, then {ϕ(x; bi)} is inconsistent if and only if it is k-inconsistent for some k.
  • If b1, b2, … is arbitrary, and {ϕ(x; bi)} is k-inconsistent, then this is witnessed in the EM-type of ⟨bi⟩i. Consequently, if c1, c2, … is an indiscernible sequence extracted from b1, b2, …, then {ϕ(x; ci)} will also be k-inconsistent, for the same k.

k-inconsistency plays a basic role in the definitions of dividing, forking, and their variants (such as thorn-forking). ::: ::: :::
