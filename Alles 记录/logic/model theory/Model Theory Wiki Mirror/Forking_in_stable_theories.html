In stable structures, there is a canonical ternary relation A↓CB on subsets A, B, C of the structure, interpreted as "A is independent from B over C." There is also a closely connected notion of a "non-forking extension" of a complete type. This is related to ↓ as follows: if C ⊂ B and a is a tuple, tp(a/B) is a non-forking extension of tp(a/C) if a↓CB, and is a forking extension of tp(a/C) if $a\operatorname{\downarrow\not{}}_{C}B$. This depends only on tp(a/B) and tp(a/C). The non-forking extensions of a complete type can be thought of as the "generic" or "canonical" extensions, in a certain sense.

The ternary independence relation and the relation of non-forking can be defined through several different approaches, described below. In practice, what is more important than the definition is the formal properties possessed by these relations. In fact, these properties can be used to axiomatize ↓. The manipulation of independence and non-forking via these axioms and other properties is called the forking calculus. Forking computations play a key role in many proofs in stability theory.

In superstable theories, forking is closely connected to ranks such as Lascar rank and Morley rank. In a general stable theory, these ranks may explode. Certain arguments that work in superstable settings can be generalized to stable theories through reformulation in terms of ↓. The relation ↓ also appears in many of the basic definitions of stability theory, such as Lascar rank, stationary types, orthogonal types, regular types, domination, and weight.

Forking calculus in stable theories can be generalized to simple theories, and more generally, rosy theories. Along the way, one or two of the axioms must be dropped, however.

Definitions[]

Work in a monster model 𝕌 of a stable theory T. Recall that the strong type of a over C, denoted stp(a/C), is the type of a over acleq(C), the algebraic closure of C in 𝕌eq. In stable theories, strong types have canonical extensions: there is a unique complete type p over 𝕌 which extends stp(a/C) and which is Aut(𝕌/acleq(C))-invariant. Denoting this unique extension by stp(a/C)|𝕌, one can make the following definition: if C ⊂ B, then tp(a/B) is a non-forking extension of tp(a/C) if and only if stp(a/B)|𝕌 = stp(a/C)|𝕌. It turns out that this definition depends only on tp(a/C) and tp(a/B).

Then for A, B, C ⊂ 𝕌, one defines A↓CB to mean that for every finite a ⊂ A, tp(a/BC) is a non-forking extension of tp(a/C).

See below for some alternative definitions.

Axioms and Basic Properties[]

(symmetry)
A↓CB is equivalent to B↓CA. That is, if A is independent from B over C, then B is independent from A over C.
(invariance)
If σ is an automorphism of 𝕌, then A↓CB ⇔ σ(A)↓σ(C)σ(B). By strong homogeneity of the monster, this can be restated as follows: if A′B′C′≡⌀ABC, then A′↓C′B′ ⇔ A↓CB. This is an unsurprising property, which would be expected of any definition which was canonical.
(monotonicity)
If A′ ⊂ A and B′ ⊂ B, then A↓CB ⇒ A′↓CB′.
(finite character)
A↓CB holds if and only if A′↓CB′ holds for all finite subsets A′ ⊂ A and B′ ⊂ B.
(transitivity)
If B1 ⊂ B2 ⊂ B3, then A↓B1B3 holds if and only if A↓B1B2 and A↓B2B3 hold. On the level of types, this means that tp(A/B3) is a non-forking extension of tp(A/B1) if and only if tp(A/B3) is a non-forking extension of tp(A/B2) and tp(A/B2) is a non-forking extension of tp(A/B1). In particular, the relation of being a non-forking extension is transitive.
(normality)
A↓CB is equivalent to A↓CBC. This means that A↓CB really depends only on the set BC, and not on B specifically.
(stationarity)
Suppose M is a model, A↓MB, A′↓MB, and A≡MA′. Then A≡MBA′. On the level of types, this means that if M ⊂ B and M is a model, then every type over M has a unique non-forking extension to B.
(existence)
A↓BB. On the level of types, this means that types are non-forking extensions of themselves.
(full existence)
Given A, B, C we can find A′≡CA such that A′↓CB. On the level of types, this means that if C ⊂ B, then every type over C has at least one non-forking extension to B.
(local character)
There is a cardinal κ such that if A is finite and C is arbitrary, then there exists C′ ⊂ C with |C′| < κ such that A↓C′C. On the level of types, this means that every type is a non-forking extension of some type over a set of size less than κ.

Some of the above properties, such as symmetry, are more readily expressed in terms of independence, while others, such as transitivity, are more readily expressed in terms of non-forking extensions. Part of the complexity of forking calculus comes from the interplay of these two notions.

The above conditions actually characterize stable theories, in the following sense: if T is a theory with monster model 𝕌 for which there exists a ternary relation ↓ satisfying the above axioms, then T is stable and ↓ is forking independence.

Forking and ranks[]

Forking is closely related to ranks such as Morley rank and Lascar rank. If R( − ) is one of Morley rank, Lascar rank, or Shelah infinity-rank, then R( − ) governs forking, in the following sense: if p(x) is a type extending q(x) and R(q) < ∞, then

  • R(p) = R(q) if and only if p(x) is a non-forking extension of q(x).
  • R(p) < R(q) if and only if p(x) is a forking extension of q(x).

Equivalently, a↓CB means that R(a/BC) = R(a/C), and $a\operatorname{\downarrow\not{}}_{C}B$ means that R(a/BC) < R(a/C).

If T has finite Lascar rank, ↓ can be defined very symmetrically as follows: A↓CB means that for every finite tuple a from A and b from B, U(ab/C) = U(a/C) + U(b/C).

Further properties[]

Aside from the axioms listed above, there are a number of other commonly used properties of forking:

  • By throwing in normality, transitivity can be slightly strengthened to the following statement: if A↓CB and A↓CBB′, then A↓CBB′.
  • Because of symmetry, there are mirror-image versions of all the above properties. For example, left transitivity says that if A↓CB and A′↓CAB, then AA′↓CB.
  • If p is a type over a model M, the non-forking extensions of p are exactly the same as the heirs of p, which are in turn the same as the coheirs of p.
  • If p is a type over a set C = acleq(C) which is algebraically closed in 𝕌eq, then types over C have unique non-forking extensions to larger sets. That is, if A↓CB and A′↓CB and A≡CA′, then A≡BCA′. Thus, strong types are stationary.
  • Whether or not A↓CB holds depends only on acl(A), acl(C), and acl(B): A↓CB ⇔ acl(A)↓acl(C)acl(B) ⇔ acl(AC)↓acl(C)acl(BC)
  • Forking is always witnessed by a formula: if tp(a/BC) is a forking extension of tp(a/C) (i.e., $a\operatorname{\downarrow\not{}}_{C}B$), then there is a BC-formula ϕ(x) ∈ tp(a/BC) such that every type extending ϕ(x) is a forking extension of its restriction to C. One says that ϕ(x) forks over C when this property holds. So, we are saying that a↓CB if and only if no formula in tp(a/BC) forks over C.
  • If A↓CB and D is some other set, we can find A′≡BCA such that A′↓CBD. On the level of types, this means that if C ⊂ B ⊂ D and p is a type on C which has some non-forking extension q to B, then q can be further extended to a type on D which is a non-forking extension of p. This fact is called extension, and is an easy consequence of transitivity and full existence.
  • If A↓CB, and D ⊂ B, then A↓CDB. This is a consequence of transitivity, as we have stated it above. This fact is usually called base monotonicity.
  • One says that a sequence {Aα}α < λ of sets is independent over C if for each α, $A_{\alpha} \downarrow_{C}A_{< \alpha},\text{~where~}A_{< \alpha} = \bigcup\limits_{\beta < \alpha}A_{\beta}.$ So a1, a2, … is independent over C if a1↓Ca2, a1a2↓Ca3, a1a2a3↓Ca4, and so on. The fundamental fact about this notion is that whether or not a sequence is independent over C does not depend on the order of the terms, that is, any permutation of an independent sequence is still independent. For example, if a1↓Ca2 and a1a2↓Ca3, we can conclude that a2a3↓Ca1.
  • Any subsequence of a C-independent sequence is still C-independent, and an infinite sequence is C-independent if and only if every finite subsequence is C-independent.
  • If I is an independent sequence of tuples over C, and I1, I2, and I3 are three pairwise disjoint subsequences, then I1↓CI2I3. For example, if a1, a2, a3 is independent over ⌀, then a1↓a2a3.
  • If {Aα}α < λ is C-independent, and b is some finite tuple, then b↓CAα must hold for all but at most κ values of α, where κ is the cardinal from the local character property.
  • Independence is preserved in non-forking extensions: if C ⊂ C′ and tp(AB/C′) is a non-forking extension of tp(AB/C) (that is, AB↓CC′, then A↓CB ⇔ A↓C′B. More generally, if A1, A2, … is a sequence, C ⊂ C′, and A1A2⋯↓CC′, then A1, A2, … is C-independent if and only if it is C′-independent.

Alternative Definitions[]

There are a number of different approaches to defining non-forking and ↓, aside from the one used above. All these definitions agree, for stable theories.

As a slight variant of the definition given above, one can proceed as follows. If a and b are finite, a↓Cb means that (stp(a/C)|𝕌) ⊗ (stp(b/C)|𝕌) = stp(ab/C)|𝕌, where ⊗ denotes the product operation on global invariant types. Alternatively, a↓Cb means that ab ⊨ (p ⊗ q)|C for two acleq(C)-definable global types p and q. Either way, A↓CB is then defined to mean that a↓Cb for every finite a ⊂ A and b ⊂ B. Then tp(a/BC) is a non-forking extension of tp(a/C) if and only if a↓CB.

Shelah's original definition of forking proceeds as follows. A formula ϕ(x; a) is said to divide over a set C if there exists a C-indiscernible sequence a1, a2, … of realizations of tp(a/C) such that $\bigwedge\limits_{i = 1}^{\infty}\phi(x;a_{i})$ is inconsistent. A formula ϕ(x; a) is said to fork over a set C if it implies a finite disjunction $\bigvee\limits_{j = 1}^{n}\psi_{j}(x;b_{j})$ of formulas (with parameters bj from 𝕌), such that each ψj(x; bj) divides over C. A type Σ(x) forks (divides) over C if some formula in it forks (divides) over C.

Then one defines A↓CB to mean that tp(a/BC) doesn’t fork over C for every finite tuple a from A. In a stable theory, it turns out that one can equivalently replace "forking" with "dividing" in this definition. Shelah’s definition of ↓ is somewhat convoluted, but turns out to be the right definition for use with unstable theories, such as simple theories, NIP theories, and NTP2 theories.

In Poizat’s Course in Model Theory a different definition of forking is given for stable theories, using the fundamental order. If p(x) is a complete type over a model, we say that a formula ϕ(x; y) is "represented" in p(x) if ϕ(x; a) ∈ p(x) for some a. If p(x) and q(x) are two complete types over models, we say that p ≤ q if every formula represented in q is also represented in p. If p is an extension of q, then p ≤ q holds, for example. This defines a pre-order on the class of types over models; the quotient poset is called the fundamental order. If p is a type over an arbitrary set A, and M ⊃ A is a model, then among the equivalence classes of extensions of p to M, there is a unique maximum in the fundamental order. This maximum is called the bound of p; it does not depend on M. Now if p and q are two complete types, and q is an extension of p, then q is a non-forking extension of p if and only if q and p have the same bound. So A↓CB means that for every finite tuple a from A, tp(a/BC) and tp(a/C) have the same bound.

In totally transcendental theories, as noted above, tp(a/BC) is a non-forking extension of tp(a/C) if and only if RM(a/BC) = RM(a/C). This can be used to define non-forking, in a totally transcendental setting. The analogous fact for Lascar rank in superstable theories cannot really be used as a definition, because Lascar rank is (usually) defined in terms of ↓.

In a general stable theory, there is still a way to define ↓ using ranks. If Δ is a finite set of formulas, and Σ(x) is a finite partial type, the local Δ-rank R(Σ(x); Δ; ω) is defined to be the Cantor-Bendixson rank of the space of Δ-types over 𝕌 which are consistent with Σ(x). If Σ(x) is an arbitrary partial type, R(Σ(x); Δ; ω) is defined to be the minimum of R(Σ0(x); Δ; ω), for Σ0(x) a finite subtype of Σ(x). In a stable theory, R(Σ(x); Δ; ω) < ∞ always holds. If p(x) is a type extending q(x) (complete types over different sets), then it turns out that p is a non-forking extension of q if and only if R(p; Δ; ω) = R(q; Δ; ω) for every finite set of formulas Δ. ::: ::: :::
