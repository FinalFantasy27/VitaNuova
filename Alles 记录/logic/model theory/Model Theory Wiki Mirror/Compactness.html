The Compactness Theorem states that if T is a collection of first-order statements and every finite subset of T is consistent, then T is itself consistent. A set of statements is consistent if it has a model.

By abuse of terminology, the following related fact is also frequently referred to as "compactness." Let M be a κ-saturated model, D ⊂ Mn be a definable set, and Σ be a collection of definable subsets of Mn, with Σ of size less than κ. If Σ covers D, then some finite subset of Σ covers D. This fact follows from the definition of κ-saturation. Compactness is used to prove the existence of κ-saturated models, however.

Common Corollaries and Uses of Compactness[]

The compactness theorem has a number of commonly-used corollaries. These corollaries are often used implicitly in proofs, and explained only as "compactness."

Lemma: Let M be a model, and Σ(x) be a consistent partial type over M. Then there is an elementary extension M′ ≽ M in which Σ(x) is realized.

Proof: Apply the Compactness Theorem to the union of the elementary diagram of M and the statements Σ(c), where c is a new constant symbol. QED

Lemma: Let T be a theory. Let ϕ(x) be a formula, and let {ψi(x) : i ∈ I} be a collection of formulas. Suppose that in every model M of T, we have

$\phi(M) \subseteq \bigcup\limits_{i \in I}\psi_{i}(M)$.

Then there is a finite subset I0 ⊂ I such that in every model M of T,

$\phi(M) \subseteq \bigcup\limits_{i \in I_{0}}\psi_{i}(M)$.

Proof: Apply compactness to the union of T and the statements

{ϕ(c)} ∪ {¬ψi(c) : i ∈ I}

where c is a new constant symbol. By assumption, this collection of statements is inconsistent, so by compactness, some finite subset is inconsistent. This yields I0. QED

Lemma: Let T be a theory, and ϕ(x) be a formula. Suppose that in every model M of T, the set ϕ(M) is finite. Then there is a number n such that |ϕ(M)| < n for every model M.

Proof: Apply compactness to the union of T and the set of sentences ψn asserting for each n that at least n elements of the model satisfy ϕ. By assumption, this is inconsistent. Consequently, there is some n such that ψn is inconsistent with T. This means exactly that |ϕ(M)| < n in every model of T. QED

The analogous statement in saturated models is the following: Lemma: Let M be a ℵ1-saturated model. Suppose that ϕ(x; y) is a formula such that for every b ∈ M, the set ϕ(M; b) is finite. Then there is a uniform bound n on the size of ϕ(M; b).

Other important and prototypical applications of compactness include the following:

  • The Upwards Löwenheim-Skolem Theorem
  • The existence of indiscernible sequences
  • The existence of κ-saturated models
  • The equivalence between the existence of Skolem functions and the property that every substructure is an elementary substructure.
  • The equivalence between elimination of imaginaries and the interdefinability of every imaginary element with a real element.
  • The statement that if every substructure of a model of T is a model of T, then T is equivalent to a set of universal statements.


Interpretation as Compactness of Stone Space[]

Let S denote the space of all complete theories (in some fixed first-order language). For each sentence ϕ, let

[ϕ] = {T ∈ S|ϕ ∈ T}

There is a natural topology on S in which the sets of the form [ϕ] are the basic open (and basic closed) subsets. The compactness theorem says exactly that S is compact with this topology.


Proofs of Compactness[]

Compactness follows easily from some forms of Gödel's Completeness Theorem. Specifically, a theory T is inconsistent if and only if ∃x : x ≠ x holds in all models of T. By the Completeness Theorem, this holds if and only if ∃x : x ≠ x can be proven from T. But proofs are finitary, so any proof must take only finitely many steps, and must use only a finite subset of T. In particular, if T proves ∃x : x ≠ x, then so does some finite subset T0 ⊂ T. So if T is inconsistent, so is a finite subset.

Compactness can alternatively be proven from Łoś's Theorem.

Proof: Let T be a collection of statements, with every finite subset of T being consistent. Let X be the set of all finite of T. For each finite subset S ⊂ T, let

XS = {T′ ∈ X|S ⊂ T′}

Note that XS ∩ XT = XS ∪ T and that XS ≠ ⌀ for any S. It follows that any finite intersection of XS's is non-empty. Therefore we can find an ultrafilter 𝒰 on X such that XS ∈ 𝒰 for every S. In other words, for each S, 𝒰 thinks that "most" elements of X contain S.

For each finite subset S of T, we can find a model MS of S, by assumption. Consider the ultraproduct

$M = \prod\limits_{S \in X}M_{S}/\mathcal{U}$

We claim that M is the desired model of T. Let ϕ be a formula in T. Then

MS ⊨ ϕ

for every S ∋ ϕ, or equivalently, for every S ∈ X{ϕ}. But

X{ϕ} ∈ 𝒰.

Consequently, the set of S such that MS ⊨ ϕ is "large" with respect to the ultrafilter 𝒰. By Łoś's Theorem, ϕ holds in the ultraproduct M. QED ::: ::: :::
