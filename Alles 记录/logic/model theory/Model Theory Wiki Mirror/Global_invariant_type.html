A global invariant type is a type over the monster M which does not split over a small set A, or equivalently, is Aut(M/A)-invariant. We say that this global type is "A-invariant" (or "invariant over A") when we want to specify the set A.

In other words, suppose M is a structure, A ⊂ M is a subset, and suppose that M is |A|+-saturated and |A|+-strongly homogeneous. Then we are interested in (complete) types p(x) on M with the following property: for every a from M and formula ϕ(x; y), whether or not ϕ(x; a) ∈ p(x) depends only on tp(a/A) (and on ϕ(x; y)). Equivalently, whenever a≡Ab, we have

ϕ(x; a) ∈ p(x) ⇔ ϕ(x; b) ∈ p(x)

By strong homogeneity, the set of b such that b≡Aa is exactly the orbit of a under the automorphism group Aut(M/A). So an equivalent condition is that for every a, every formula ϕ(x; y), and every σ ∈ Aut(M/A), we have

ϕ(x; a) ∈ p(x) ⇔ ϕ(x; σ(a)) ∈ p(x).

This is in turn equivalent to the condition that p = σ*p for every σ ∈ Aut(M/A), i.e., that p is invariant under Aut(M/A).

Global invariant types are essentially the same thing as what Poizat calls "special sons" in his Course in Model Theory.

A somewhat prototypical example of a global invariant type is the generic type of a geometrically irreducible variety in ACF. Let K be a field, and embed K in some very saturated and strongly homogeneous monster model M of ACF. If V ⊂ 𝔸Kn is a geometrically irreducible affine variety, then V is irreducible over M, and we can talk about the generic type p(x) ∈ Sn(M). This is the type which says that the n-tuple x lies on V, but not on any strictly smaller M-definable closed subvarieties of V. This type is K-invariant, i.e., Aut(M/K)-invariant. This is fairly clear by symmetry considerations: if σ ∈ Aut(M/K), then σ sends the variety V to itself, and sends proper subvarieties of V to proper subvarieties of V. Since p is characterized as being the unique type over M which lies on V but not on any proper subvarieties, σ(p) must be the same type as p.

We should also point out that coheirs are invariant types: if a global type p(x) is a coheir of its restriction to a small model M, then p(x) is M-invariant.

Definable types are another source of global invariant types. (In fact, in stable theories, definable types and invariant types are the same thing.)

Existence of global invariant types[]

Work inside a monster model 𝔐 of some theory.

Theorem: Let M be a small model (i.e., small elementary substructure of 𝔐), and p(x) be a complete type over M. Then there is at least one global M-invariant type extending p.

Proof: Let Σ(x) be the (big) partial type over 𝔐 consisting of the statements

ϕ(x; a) ↔︎ ϕ(x; b)

for every formula ϕ(x; y), and every a, b from 𝔐 such that a≡Mb (i.e., such that a and b have the same type over M).

Note that Σ(x) is satisfied by every element of M. Indeed, if c ∈ M and a≡Mb, then a≡cb so ca≡⌀cb, and therefore

𝔐 ⊨ ϕ(c; a) ⇔ 𝔐 ⊨ ϕ(c; b)

for every formula ϕ(x; y).

Now any finite subset p0(x) ⊂ p(x) is satisfied by some tuple c from M. This tuple satisfies p0(x) ∪ Σ(x). It follows that p0(x) ∪ Σ(x) is consistent for every finite subset p0(x) ⊂ p(x). Therefore p(x) ∪ Σ(x) is a consistent partial type over the monster 𝔐.

Let q(x) be any complete type over the monster 𝔐 extending p(x) ⊂ Σ(x). Then q(x) is our desired global M-invariant type. Indeed, if a and b are two tuples from 𝔐 having the same type over M, then

q(x) ⊢ ϕ(x; a) ↔︎ ϕ(x; b)

because q(x) extends Σ(x). But because q(x) is a complete type, it follows that

ϕ(x; a) ∈ q(x) ⇔ ϕ(x; b) ∈ q(x).

Given that this holds for any formula ϕ(x; y) and any two tuples a, b having the same type over M, it follows that q(x) is M-invariant, for reasons explained above.

QED.

From this we see that non-trivial examples of global invariant types exist in any theory.

Invariant types as a variation on Definable types[]

A global type p(x) is A-definable if for every formula ϕ(x; y), there is a formula ψ(y) over A such that

ϕ(x; b) ∈ p(x) ⇔ 𝔐 ⊨ ψ(b),

for every b in M. We think of the map ϕ(x; y) ↦ ψ(y) as a "definition schema." The formula ψ(y) is called the "ϕ-definition of p." If p is definable, then it is always A-definable over some small set A.

Now if b≡Ab′, then

ϕ(x; b) ∈ p(x) ⇔  ⊨ ψ(b) ⇔  ⊨ ψ(b′) ⇔ ϕ(x; b′) ∈ p(x),

where the middle  ⇔  comes from the fact that ψ(y) is a formula over A. Thus p is A-definable; A-definable types are A-invariant. The notion of "A-invariant type" can be seen as a generalization of "A-definable type" in which the ϕ-definition of p is no longer given by a formula over A, but by a set of complete types over A. That is, there is some subset X ⊂ S(A) such that for every b,

ϕ(x; b) ∈ p(x) ⇔ tp(b/A) ∈ X.

If X were the clopen set associated with some A-formula ψ(y), then this would mean that

ϕ(x; b) ∈ p(x) ⇔  ⊨ ψ(b).

An A-definable type p(x) over a model M has a unique global extension q(x) which is A-definable. Essentially, the definition schema for q(x) must be the same as the definition schema for p(x). Something similar happens with invariant types:

Theorem: If M is |A|+-saturated and |A|+-strongly homogeneous, then each A-invariant type on M has a unique global extension which is A-invariant.

Proof: Let q(x) be the set of all formulas ϕ(x; b), where b is a tuple from the monster 𝔐 such that b≡Ac for some tuple c from M such that ϕ(x; c) ∈ p(x). We claim that q(x) is a consistent type, it is a complete type over the monster, that it is A-invariant, and that it is the only global A-invariant type.

First of all, we check consistency. Suppose for the sake of contradiction that

{ϕ1(x; b1), …, ϕn(x; bn) ⊂ q(x)

is inconsistent. By definition of q(x), we can find c1, …, cn such that bi≡Aci for each i, and ϕi(x; ci) ∈ p(x).

Now because M is |A|+-saturated, we can find d1, …, dn in M such that

b1b2⋯bn≡Ad1d2⋯dn.

Then for each i,

di≡Abi≡Aci.

By strong homogeneity, some σ ∈ Aut(M/A) sends di to ci. By invariance of p(x),

ϕi(x; di) ∈ p(x) ⇔ ϕi(x; ci) ∈ p(x).

So ϕi(x; di) ∈ p(x) for i = 1, …, n. Now by assumption

$\vDash \neg\exists x:\bigwedge\limits_{i = 1}^{n}\phi_{i}(x;b_{i})$.

Since the bi's collectively have the same type as the di's,

$\vDash \neg\exists x:\bigwedge\limits_{i = 1}^{n}\phi_{i}(x;d_{i})$.

But we just claimed that ϕi(x; di) ∈ p(x) for each i, so p(x) is inconsistent, contradicting the fact that it is a complete type.

Next, we check that q is a complete type. Let ϕ(x; y) be a formula and b be a tuple from the monster. By saturation of M, we can find some c in M such that b≡Ac. Because p is complete, either ϕ(x; c) or ¬ϕ(x; c) is in p(x). Then by definition of q(x), either ϕ(x; b) or ¬ϕ(x; b) is in q(x). So q is complete.

Next we check A-invariance. Suppose b≡Ab′ for b, b′ in the monster. By saturation of M, we can find c in M realizing tp(b/A) = tp(b′/A). Then by definition of q, for each formula ϕ(x; y)

ϕ(x; b) ∈ q(x) ⇔ ϕ(x; c) ∈ p(x) ⇔ ϕ(x; b′) ∈ q(x).

As explained above, this shows that q is A-invariant.

Finally, suppose r were some other A-invariant global type extending p. Then for any b in the monster and any formula ϕ(x; y), we have

ϕ(x; b) ∈ r(x) ⇔ ϕ(x; c) ∈ r(x) ⇔ ϕ(x; c) ∈ p(x) ⇔ ϕ(x; c) ∈ q(x) ⇔ ϕ(x; b) ∈ q(x)

where c is any realization of tp(b/A) in M.

QED.

The tensor product of global invariant types[]

Work in a monster model 𝔐. Recall the following (standard) notation:

  • If p is a global type, and A is a small subset, then p ↾ A denotes the restriction of p to A, i.e., the complete type over A implied by p.
  • If p is a type and a is a tuple, a ⊨ p means that a satisfies p.
  • If C is a set and a is a tuple, Ca or aC denotes C ∪ {a}. In other words, union or concatenation is written like multiplication.

Lemma: Let A be a small subset, let q(x) be a (complete) type over A, and let p(y) be a global A-invariant type. Then

S = {(a, b) from the monster : b ⊨ q and a ⊨ p ↾ Ab}

is the set of realizations (in the monster) of some complete type r(x; y) over A.

Proof: First of all, S is non-empty, because q and p are consistent types (and the monster is saturated).

Next, any two tuples in S have the same type over A. Indeed, suppose that (a, b), (a′, b′) ∈ S. Then b and b′ both realize q, so there exists some automorphism σ over A sending b to b′. Because p is A-invariant, σ(p) = p. The fact that a realizes p ↾ Ab implies therefore that σ(a) realizes

σ(p) ↾ Aσ(b) = p ↾ Ab′ = tp(a′/Ab′).

Thus σ(a)≡Ab′a′. But then

ab≡Aσ(a)σ(b) = σ(a)b′≡Aa′b′,

so (a, b) and (a′, b′) have the same type over A, as claimed.

Finally, suppose that (a, b) is in S, and a′b′≡Aab. Then b′≡Ab, so b′ realizes q. Also, there is some automorphism σ fixing A, and sending b to b′. Then σ fixes p, so

a ⊨ p ↾ Ab ⇒ σ(a) ⊨ p ↾ Ab′.

Meanwhile,

a′b′≡Aab≡Aσ(a)σ(b) = σ(a)b′,

so

a′≡Ab′σ(a).

Then since σ(a) realizes p ↾ Ab′, so does a′. Now we know that

b′ ⊨ q and a′ ⊨ p ↾ Ab′.

So (a′, b′) ∈ S, as claimed.

(A shorter way to see this is to argue from symmetry that S must be A-invariant, because it is defined using only A, a type over A, and an A-invariant global type. So any symmetry which fixes A must certainly send S to itself. Then if (a, b) ∈ S and (a′, b′)≡A(a, b), some automorphism over A sends (a, b) to (a′, b′), and therefore (a′, b′) is in S.)

At any rate, we have shown that S is non-empty, any two elements have the same type over A, and that any tuple having the same type over A as someone in S is itself in S. Therefore, S is exactly the set of realizations of some complete type r over A.

QED.

Theorem/Definition: Let p(x) and q(y) be two global A-invariant types. Then there is a unique global type (p ⊗ q)(x, y) with the following property: for every small set C containing A, and every a and b from the monster, the following are equivalent:

  • (a, b) ⊨ (p ⊗ q) ↾ C
  • b ⊨ q ↾ C and a ⊨ p ↾ Cb

The type p ⊗ q is A-invariant. Moreover, p ⊗ q does not depend on the choice of A (i.e., it remains the same if we replace A with a bigger set.)

Proof: Note that for any small C containing A, p is C-invariant, because Aut(𝔐/C) ⊂ Aut(𝔐/A). By the Lemma, we can find a complete type rC(x; y) over C such that the following are equivalent for a and b tuples from the monster of the appropriate lengths:

  • (a, b) ⊨ rC
  • b ⊨ q ↾ C and a ⊨ p ↾ Ca.

We claim that if C ⊂ D, then rD extends rC. Equivalently, if (a, b) ⊨ rD, then (a, b) ⊨ rC. But this is clear because

b ⊨ q ↾ D ⇒ b ⊨ q ↾ C

and

a ⊨ p ↾ Da ⇒ a ⊨ p ↾ Ca.

Let p ⊗ q be the union of the rC's as C ranges over small supsersets of A. From what we have just seen, p ⊗ q is a consistent type, hence a complete global type. It has the desired characterization that for any small set C containing A,

  • (a, b) ⊨ (p ⊗ q) ↾ C

if and only if

  • b ⊨ q ↾ C and a ⊨ p ↾ Cb.

This uniquely characterizes the type. Moreover, p ⊗ q must be A-invariant, by symmetry. (It is defined only using A-invariant entities.)

Finally, let us verify that the choice of A does not matter. For small B ⊇ A, let p⊗Bq denote the B-invariant tensor product of p and q viewed as B-invariant types. The conditions which uniquely characterize p⊗Aq are clearly as strong as the conditions which uniquely characterize p⊗Bq, so the two must be equal:

p⊗Aq = p⊗Bq.

Now if A and B are two different small sets over which p and q are invariant, then

p⊗Aq = p⊗A ∪ Bq = p⊗Bq.

QED.

Now if p(x) and q(y) are two invariant types (over some small sets), then we can uniquely define the tensor product (p ⊗ q)(x, y), by finding some A over which both p and q are invariant. (For example, if p is B-invariant and q is C-invariant, then any small A ⊇ B ∪ C works.)

The tensor product (p ⊗ q)(x, y) is usually denoted p(x) ⊗ q(y).

The Example of ACF[]

The tensor product operation on global invariant types generalizes the notion of taking the cartesian product of two varieties, in the following sense. Work in a monster model of ACF. Let V and W be two irreducible affine varieties, and let p and q denote the corresponding generic types, over the monster model. Then p ⊗ q is the generic type of V × W. Let's prove this.

Claim: Let V be an irreducible variety. Let p be the global generic type of V. Let K be any (small) field over which V is defined. Then a tuple a (from the monster) realizes p ↾ K if and only if a is on V and tr.deg(a/K) = dim(V).

Proof: First suppose that a ⊨ p ↾ K. The statement x ∈ V is certainly a formula over K, so it is in p ↾ K, and therefore a ∈ V. This ensures that tr.deg(a/K) ≤ dim(V). If the equality was strict, then a would lie on some K-definable variety W of dimension less than dim(V). (Of course W might not be irreducible.) Then V ∩ W is a K-definable proper closed subvariety of V. The type p(x) certainly contains the statement x ∉ V ∩ W, and since this statement is over K, it is in p ↾ K. Therefore a ∉ V ∩ W, a contradiction.

For the converse, let $\overset{¯}{K}$ denote the algebraic closure of K. This is a model of ACF, so an elementary substructure of the monster (by model completeness of ACF). The type $p(x) \upharpoonright \overset{¯}{K}$ says (among other things) that x is on V and not on any strictly smaller $\overset{¯}{K}$-definable subvarieties of V, if not more. But by the characterization of complete types over models of ACF, $p \upharpoonright \overset{¯}{K}$ can be nothing but the generic type of V, as one would expect.

Now if a is on V and tr.deg(a/K) = dim(V), then a is on V, and not on any strictly smaller $\overset{¯}{K}$-definable subvarieties of V. (Indeed, if a is on $\overset{¯}{K}$-definable W ⊊ V, then dim(W) < dim(V) and

$\text{tr.deg}(a/K) = \text{tr.deg}(a/\overset{¯}{K}) \leq \text{dim}(W) < \text{dim}(V) = \text{tr.deg}(a/K)$,

a contradiction.) So a realizes the generic type of V over $\overset{¯}{K}$, i.e., a realizes $p \upharpoonright \overset{¯}{K}$. It certainly then also realizes the restriction of this to K, which is p ↾ K.

The claim is proven.

Theorem: Let V and W be irreducible varieties with global generic types p(x and q(y). Then p(x) ⊗ q(y) is the generic type of V × W.

Proof: Let K be a field over which V and W are defined. So p, q, and p ⊗ q are all K-invariant. Let r be the global generic type of V × W. The type r is also K-invariant.

For any small L ⊇ K, and tuples a and b of the appropriate lengths, we have the following series of equivalent statements

  • (a, b) ⊨ (p ⊗ q) ↾ L
  • b ⊨ q ↾ L and a ⊨ p ↾ L(b) (where L(b) is the field obtained by adjoining b to L. Note that since L(b) and L ∪ {b} are interdefinable, it is harmless to use L(b) instead of L ∪ {b}. This is also why we are able to focus on small subfields, rather than on small subsets.)
  • b ∈ W and tr.deg(b/L) = dim(W) and a ∈ V and tr.deg(a/L(b)) = dim(V).
  • (a, b) ∈ V × W and tr.deg(b/L) = dim(W) and tr.deg(a/L(b)) = dim(V)
  • (a, b) ∈ V × W and tr.deg((a,b)/L) = dim(V) + dim(W). [This is equivalent to the previous statement because of the identity tr.deg((a,b)/L) = tr.deg(a/L(b)) + tr.deg(b/L) coupled with the fact that a ∈ V forces tr.deg(a/L(b)) ≤ dim(V) and that b ∈ W forces tr.deg(b/L) ≤ dim(W).]
  • (a, b) ∈ V × W and tr.deg(ab/L) = dim(V × W).
  • (a, b) ⊨ r ↾ L. [This is equivalent to the previous statement by the Claim.]

So we see that p ⊗ q and r have the same restriction to any small subfield. Therefore, they are equal.

QED.

Associativity[]

The tensor-product operation on invariant types is associative, in the sense that ((p ⊗ q) ⊗ r) is the same as (p ⊗ (q ⊗ r)). This follows by unwinding the definitions.

Indeed, if C is any small set over which p, q, and r are invariant, and if a, b, and c are appropriately-sized tuples, then the following conditions are equivalent:

  • (a, b, c) ⊨ ((p ⊗ q) ⊗ r) ↾ C
  • c ⊨ r ↾ C and (a, b) ⊨ (p ⊗ q) ↾ Cc
  • c ⊨ r ↾ C and b ⊨ q ↾ Cc and a ⊨ p ↾ Cbc.
  • (b, c) ⊨ (q ⊗ r) ↾ C and a ⊨ p ↾ Cbc
  • (a, b, c) ⊨ (p ⊗ (q ⊗ r)) ↾ C.

Each of these conditions is equivalent to the next one, by the definition of ⊗.

Since associativity holds, we can write p(x) ⊗ q(y) ⊗ r(z) unambiguously.

In general, a realization of p1(x1) ⊗ ⋯ ⊗ pn(xn)'s restriction to C is a sequence a1, a2, …, an such that

  • an realizes pn restricted to C.
  • an − 1 realizes pn − 1 restricted to Can.
  • an − 2 realizes pn − 2 restricted to Canan − 1.

And so on, down to

  • a1 realizes p1 restricted to Ca1.

In the example of ACF, associativity means that the generic type of (V1 × V2) × V3 is the same as the generic type of V1 × (V2 × V3), which is clear.

[]{#.28Failure_of.29_Commutativity}[(Failure of) Commutativity]{#(Failure_of)_Commutativity .mw-headline}[]

Two global invariant types p(x) and q(y) are said to commute if p(x) ⊗ q(y) = q(y) ⊗ p(x). In other words,

(a, b) ⊨ p ⊗ q ↾ C ⇔ (b, a) ⊨ q ⊗ p ↾ C.

(This is not the same as saying that p ⊗ q is q ⊗ p; the variables must be kept track of.)

Ideally, commutativity would always happen. This is the case in stable theories, and in NIP theories when one of the types is generically stable. In the example of ACF, this reflects the fact that if (a, b) is a generic point on V × W, then (b, a) is a generic point on W × V.

Here is an example of failure of commutativity. In the theory of dense linear orders, let p be the definable 1-type at +∞. This is Ø-invariant. If C is a small subset of the monster, then (a, b) ⊨ p(x) ⊗ p(y) ↾ C if b > C (i.e., b is bigger than every element of C) and a > b. But of course this is different from (b, a) ⊨ p(x), which implies that b > a > C. So p does not even commute with itself.

Morley Sequences of Invariant Types[]

If p is a global A-invariant type, a Morley sequence in the type p is a sequence a1, a2, … where

ai ⊨ p ↾ Aa1a2⋯ai − 1

for each i. This definition easily generalizes to Morley sequences of any ordinal length.

Theorem: A Morley sequence in the type p is indiscernible over A'. Any two Morley sequences in p have the same type over A (assuming they have the same length).

Proof: Let a1, a2, … be a Morley sequence. Note that if i1 < ⋯ < in then for 1 ≤ k ≤ n, aik realizes p ↾ Aai1⋯aik − 1, because {ai1, …, aik − 1} ⊆ {a1, …, aik}. It follows that

(aik, aik − 1, …, ai1) ⊨ p(x1) ⊗ ⋯ ⊗ p(xn) ↾ A

Consequently, if j1 < ⋯jn is some other ascending sequence, then

(aik, aik − 1, …, ai1) and (ajk, ajk − 1, …, aj1) both realize p ⊗ n ↾ A.

Therefore, we have indiscernibility. Also, if b1, b2, … is some other Morley sequence, then for each n,

(an, an − 1, …, a1) and (bn, bn − 1, …, b1) both realize p ⊗ n ↾ A.

So a1⋯an≡Ab1⋯bn. Since this happens for all n, the two Morley sequences have the same type over A. (This argument can easily be modified to handle the case of Morley sequences of length more than ω.)

QED.

Morley sequences of this type play a key role in elementary stability theory, where they can be used to prove the definability of types in stable theories.

Also, we can use this kind of Morley sequence to prove the existence of non-trivial indiscernible sequences.

Corollary: Let M be a structure with an infinite total ordering < (one one of the sorts). Then there is an elementary extension of M in which there is a strictly increasing indiscernible sequence a1 < a2 < ⋯.

Proof: Let 𝔐 be a monster model of the complete theory of M. Since M (or its totally ordered sort) is infinite, there is some complete type p over M which lives in the ordered sort, but is not realized in M (i.e., is not constant). Let q be a global M-invariant extension of p, which exists for reasons explained above. If q were realized in 𝔐 then q(x) would be a constant type of the form x = a for some a from 𝔐. Then by M-invariance, a would need to be fixed by every automorphism which fixes M. This would force a to be in the definable closure of M, which is M. So q(x) includes the statement x = a, and so does p(x), contradicting the assumption that p is not a constant type.

So q is not a constant type. Then for any small set C and any c ∈ C, the formula x ≠ c is in q(x) ↾ C.

In particular, if a1, a2, … is a Morley sequence for q, then ai ⊨ q ↾ Ma1…ai − 1, so ai ∉ {a1, …, ai − 1}. In other words, no two elements of the Morley sequence are equal. Also, the sequence is M-indiscernible, hence indiscernible (over the empty set).

Because the ordering is total, either a1 < a2 or a1 > a2. In the former case, indiscernibility implies that ai < ai + 1 for every i, so the sequence is increasing and we are done. Otherwise, we can extract an indiscernible sequence of the same Ehrenfeucht-Mostowski type, but indexed by the negative integers

…, b − 3, b − 2, b − 1.

As part of the EM type, we see that

⋯ > b − 3 > b − 2 > b − 1

Then

b − 1 < b − 2 < b − 3 < ⋯

is our desired increasing indiscernible sequence.

In more detail (without mentioning EM types), consider the infinitary partial type Σ(x1, x2, x3, …) consisting of the statements asserting indiscernibility of (x1, x2, …) together with x1 < x2. Then Σ(x1, x2, …) is consistent, because if Σ0(x1, x2, …) is a finite subset, then Σ0 only mentions x1, …, xn for some n, and then

(an, an − 1, …, a1, garbage) ⊨ Σ0(x1, …, xn, …)

Any realization of the consistent type Σ(x1, x2, …) will yield our desired indiscernible sequence.

Having found the increasing indiscernible sequence, we can pass to a small model if we like, or just stick with the monster 𝔐 as our elementary extension of M.

QED.

From this, it is not hard to prove that indiscernible sequences can be extracted in general.

Corollary: Let M be a structure, and let {ai}i ∈ I be any infinite sequence (with I an ordered set). Then we can find an elementary extension of M in which there is an indiscernible sequence realizing the EM type of {ai}i ∈ I.

Proof: Consider the auxiliary structure M′ obtained by adding I as a new sort, and adding symbols for the total ordering on I and a function symbol f for the map i ↦ ai. By the previous Corollary, we can find an elementary extension N′ ≽ M′ containing an increasing indiscernible sequence α1 < α2 < ⋯ in the new I sort. Then f(α1), f(α2), … is certainly indiscernible in N′. Let N be the reduct of N′ to the original language (dropping the extra I sort). Then N is an elementary extension of M. Also, f(α1), f(α2), … is certainly indiscernible in N.

Finally, we check that this indiscernible sequence satisfies the correct EM type. Suppose that ϕ(x1, …, xn) is a formula in the EM type of {ai}i ∈ I. This means that

M ⊨ ϕ(ai1, …, ain) whenever i1 < …in.

But then

M′ ⊨ ∀x1, …xn : x1 < x2 < ⋯xn → ϕ(f(x1), …, f(xn)).

Of course the same thing must hold in N′, which means that for any j1 < j2 < … < jn,

N′ ⊨ ϕ(f(αj1), f(αj2), …, f(αjn),

because αj1 < ⋯ < αjn.

Then since ϕ was a formula in the original language,

N ⊨ ϕ(f(αj1), …, f(αjn)).

As this holds for any j1 < j2 < ⋯ < jn, the sequence f(α1), f(α2), … really does satisfy the correct EM type.

QED. ::: ::: :::
