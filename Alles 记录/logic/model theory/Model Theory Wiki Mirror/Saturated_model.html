In this article, "type" will by default mean "complete type," rather than "partial type." Also, types will be finitary (n-types) rather than finitary, by default.

Roughly speaking, a structure is saturated if all types over it are realized. This is practically impossible, so instead we only consider types over small sets. If κ is an infinite cardinal, a structure M is said to be κ-saturated if every type over a subset A ⊆ M with |A| < κ is realized in M. It turns out that an equivalent condition is that every 1-type over a subset of size less than κ is realized.

The structure M is said to be saturated if it is |M|-saturated: every type over a subset of smaller cardinality than M is realized.

A closely related concept is strong homogeneity. A structure M is said to be κ-strongly homogeneous if whenever a and b are possibly infinite tuples from M having the same type over the empty set Ø, and a and b have length less than κ, then some automorphism σ of M sends a to b. Equivalently, every partial elementary map on M of size less than κ can be extended to an automorphism of M. Warning: Some authors use the term "κ-homogeneous" rather than "κ-strongly homogeneous."

In stability theory, one frequently works with a "monster model", by which one means a κ-saturated and κ-strongly homogeneous model of the theory for some cardinal κ much bigger than any cardinals we expect to run into.

Saturation as a kind of Compactness[]

Saturation can be viewed as an analogue of compactness:

Theorem: Let M be κ-saturated. Let D be some definable subset of M (or of a power of M). Let {Ei}i ∈ I be some infinite family of definable sets, with

$D \subseteq \bigcup\limits_{i \in I}E_{i}$

If the family has size less than κ, i.e., if |I| < κ, then D is covered by some finite collection of the Ei's.

Proof: Let Σ(x) be the partial type over M asserting that x is in D but in none of the Ei's. Then Σ(x) consists of fewer than κ formulas, so it is a partial type over a subset A ⊆ M with |A| < κ. If Σ(x) is inconsistent, then it is finitely inconsistent. Therefore there exists i1, …, in such that

$M \vDash \forall x:(x \in D)\rightarrow\left( {\bigvee\limits_{j = 1}^{n}x \in E_{i_{j}}} \right)$

or equivalently

$D \subseteq \bigcup\limits_{j = 1}^{n}E_{i_{j}}$. Then we are done.

Otherwise, Σ(x) is consistent. Let p(x) be a complete type over A extending Σ(x). By saturation, p(x) is realized in M, hence so is Σ(x). But a realization of Σ(x) is a point which is in D but in no Ei, contradicting the hypotheses.

QED.

Existence of saturated and strongly homogeneous models[]

A basic fact about κ-saturated and κ-strongly homogeneous structures is that enough of them exist:

  • Every consistent theory T has a model which is both κ-saturated and κ-strongly homogeneous.
  • In fact, if M is any structure and κ is any infinite cardinal, then M has an elementary extension which is κ-saturated and κ-strongly homogeneous.

Unfortunately it is hard to guarantee the existence of saturated models of set-size. Here are some results in these directions:

  • Countable ultrapowers of countable structures in countable languages are always ℵ1-saturated, and typically have cardinality 2|ℵ0|. If the continuum hypothesis holds, it follows that countable theories have saturated models.
  • More generally, if the generalized continuum hypothesis holds, there is some way to construct saturated models using ultrapowers, if I recall correctly.
  • If M has size less than some inaccessible cardinal κ, then M has a saturated elementary extension of size κ. In particular, if there are a proper class of inaccessible cardinals, then every structure has a saturated elementary extension.
  • Stable theories have saturated models. In particular, if a structure M is stable, then M has a saturated elementary extension.

An easy work around this problem is to take a conservative extension of ZFC. Take for example the BGC (Bernays-Goedel-Global Choice) were we have class-size elements. In that we can construct a monster model of class-size. For details see the Tent and Ziegler book.  Now if one wants to realise global-types of this class size monster model in a bigger, one can construct a monster of size "class of all classes". But this has to be done in a conservative of BGC again.

ZFC cannot show that every consistent first-order theory has a saturated model (Hodges 1993 p.506)

Properties of Saturated Models[]

Saturated models are strongly homogeneous:

Theorem: If M is saturated (i.e., |M|-saturated), then M is |M|-strongly homogeneous.

Saturated structures are determined up to isomorphism by their complete theory and cardinality:

Theorem: If T is a complete theory, any two saturated models of T of the same cardinality are isomorphic. Equivalently, if M and N are two elementarily equivalent saturated structures, of the same cardinality, then M and N are isomorphic.

Strongly Homogeneous Structures[]

The main use of strong homogeneity lies in the following basic facts:

Theorem: Let M be a κ-strongly homogeneous structure. Let A ⊆ M have size less than κ. Let a and b be two tuples of length less than κ. Then a≡Ab (i.e., tp(a/A) = tp(b/A)) if and only if there is an automorphism σ ∈ Aut(M/A) sending a to b. Here Aut(M/A) is the group of automorphisms of M fixing A pointwise.

Proof: If some σ ∈ Aut(M/A) sends a to b, then for any formula ϕ(x; c) over A, we certainly have

M ⊨ ϕ(a; c) ⇔ M ⊨ ϕ(σ(b); σ(c)) ⇔ M ⊨ ϕ(b; c),

by symmetry. This does not require strong homogeneity.

Conversely, suppose a≡Ab. Then Aa≡⌀Ab, where we are using the standard notation Aa to denote the concatenation of A and a. At any rate, there is then a partial elementary map f from A ∪ a to A ∪ b fixing A pointwise and sending a to b. By strong homogeneity, this can be extended to an automorphism in Aut(M/A).

QED.

Theorem: Let M be a κ-saturated and κ-strongly homogeneous structure. Let A ⊆ M be a subset of size less than κ, and a be a finite tuple from M. Then a is in the definable closure of A if and only if a is fixed by Aut(M/A). Similarly, a is in the algebraic closure of A if and only if a has finite orbit under Aut(M/A).

See the article on definable and algebraic closure for the proof. One uses κ-saturation to verify that

  • a ∈ dcl(A) if and only if a is the unique realization of tp(a/A) in M.
  • a ∈ acl(A) if and only if tp(a/A) has finitely many realizations in M.

Then one uses the previous Theorem to conclude that the set of realizations of tp(a/A) is exactly the orbit of a under Aut(M/A).

More generally, the definable closure part of this Theorem remains true without the assumption that a is finite, but merely that a has size less than κ.

Proofs[]

[]{#.CE.BA-Saturated_Models_Exist}κ-Saturated Models Exist[]

Lemma 1: Let M be a structure. Then there exists an elementary extension of M in which every type over M is realized.

Proof: Let T be the union of the elementary diagram together with the statements

{ϕ(cp; a)|ϕ(x; a) ∈ p, p ∈ Sn(M), n < ω}

where we have added a new constant symbol cp for each type p over M. If T0 is a finite subset of T, then M can be made into a model of T0 by choosing the cp appropriately. Indeed, for each type p, the set of statements in T0 concerning cp is a finite subset of p, and is therefore satisfiable in M.

So by compactness, T has a model N. Since T contains the elementary diagram of M, N ≽ M. Also, for each type p over M, N contains an element cp satisfying p, by choice of T. QED.


Theorem: Let M be a structure and κ be an infinite cardinal. Then M has a κ-saturated elementary extension.

Proof: A κ+-saturated elementary extension of M will certainly be κ-saturated, so replacing κ with κ+, we may assume that κ is regular.

Build an ascending chain {Mα}α < κ of structures inductive as follows:

  • M0 = M
  • Mα + 1 is an elementary extension of Mα in which all types over Mα are realized. We can find such an elementary extension by the Lemma.
  • If λ is a limit ordinal, $M_{\lambda} = \bigcup\limits_{\alpha < \lambda}M_{\alpha}$.

On easily proves by induction on α that Mα ≽ Mβ for all β < α, using the Tarski-Vaught Theorem at the limit ordinals. So this is an elementary chain of models. Let N be the union of this elementary chain. Again, by Tarski-Vaught, N is an elementary extension of every Mα. In particular, it is an elementary extension of M0 = M.

It suffices to show that N is κ-saturated. Let A be a subset of N of size less than κ. Because κ is regular, A ⊂ Mα for some α < κ. Now every type over A can be extended to a (complete) type over Mα. Since every type over Mα is realized in Mα + 1, so is every type over A. Consequently, every type over A is realized in N. Thus 'N is κ-saturated. QED

Partial Elementary Maps[]

Recall that if M, N are two structures, a partial elementary map from M to N is a map f : A -> N for some subset A ⊆ M with the property that for every tuple a from A and every formula ϕ(x),

M ⊨ ϕ(a) ⇔ N ⊨ ϕ(f(a))

Partial elementary maps are always injective (consider the case where ϕ(x, y) is the formula x = y). The inverse of a partial elementary map is always a partial elementary map.

The notion of a partial elementary map is closely related to the notion of type. Specifically, if α is an (infinite) tuple enumerating A, then f is a partial elementary map if and only if α and f(α) have the same type over the empty set, i.e.,

α≡⌀f(α).

In particular, if α and β are tuples from M and N having the same type, then there is a partial elementary map mapping αi to βi for each i in the index set.

If f : A -> B is a partial elementary map from A ⊆ M onto B ⊆ N, then we can push forward types from A to B. Specifically, let p be a type on A. The push-foward f*p(x) is the type on B given as follows:

f*p(x) = {ϕ(x; f(a))|a ∈ A, ϕ(x; a) ∈ p(x)}

The fact that the type f*p is consistent follows from the fact that f is a partial elementary map. This definition even makes sense when p is an infinitary type.

Observation: Let a and b be possibly infinite tuples in M and N respectively, having the same type over the empty set. So there is an elementary map f sending a coordinatewise to b. If c is a (possibly infinite) tuple from M, and p = tp(c/a), then a tuple d from N realizes f*p if and only if

ac≡⌀bd,

i.e., we can extend the partial elementary map f to a ∪ c by mapping c to d. (Here, we are using ac to denote the concatenation of a and c, and similarly for bd.)

Proof: This really follows by unwinding the definitions. The type p is the set

{ϕ(x; a)|M ⊨ ϕ(c; a)}

So f*p is

{ϕ(x; b)|M ⊨ ϕ(c; a)}

And in particular, d realizes f*p if

M ⊨ ϕ(c; a) ⇒ N ⊨ ϕ(d; b)

for all formulas ϕ(x; y). Replacing ϕ(x; y) with its negation, we get the implication in the other direction. So d realizes f*p if and only if

M ⊨ ϕ(c; a) ⇔ N ⊨ ϕ(d; b)

for all formulas ϕ. This is equivalent to

ac≡⌀bd

QED.

From this, we conclude:

Lemma 2: Let M and N be structures, and f be a partial elementary map from a subset A ⊆ M onto a subset B ⊆ N. Suppose c is a singleton from A, and every 1-type over B is realized in N. Then we can extend f to a partial elementary map from A ∪ {c} to (some subset of) N.

Proof: Let p = tp(c/A). This is a 1-type because c is a singleton. The pushforward f*p is also a 1-type; by assumption it is realized by some d in N. Then by the Observation, we can extend f to A ∪ {c} by declaring f(c) = d.

QED.

From 1-types to n-types and infinitary types[]

Theorem: Let M be a structure, and κ be an infinite cardinal. The following are equivalent:

  • For every subset A ⊆ M of size less than κ, every 1-type over A is realized in M.
  • For every subset A ⊆ M of size less than κ, every n-type over A is realized in M, for all n.
  • For every subset A ⊆ M of size less than κ, and every (complete) infinitary type p(x) over A, with x a tuple of length at most κ, p is realized in M.

Proof: Each successive condition is clearly stronger than the previous ones, so it suffices to prove the third condition from the first.

Assume that M has the first property, so every 1-type over a subset of M of size less than κ is realized in M. Let A ⊆ M have size less than κ. Let p(x) be a finitary or infinitary type over A. We may assume that the tuple of variables x is indexed by some cardinal number λ ≤ κ. Let N be an elementary extension of M in which p(x) is realized by some tuple c. We want to find a tuple d in M such that

c≡Ad

or equivalently

Ac≡⌀Ad.

The identity map on A is a partial elementary map from a subset of N to M, and we basically want to extend the domain of this partial elementary map from A to Ac.

Let c < α denote the initial segment of the tuple c consisting of cβ for β < α.

Inductively build an increasing sequence of partial elementary maps

fα : Ac < α → M

as follows

  • f0 is the identity map from A to A
  • Given fα, apply Lemma 2 above to extend the domain of this map to the element cα, yielding

fα + 1 : Ac < αcα → M

Here we are using the fact that Ac < α is a set of size less than κ, so that every 1-type over f(Ac < α) is realized in M, by assumption.

  • If α is a limit ordinal, let fα be the union of fβ for β < α. An easy exercise shows that this is a partial elementary map. (A function f is a partial elementary map if and only if its restriction to every finite set is a partial elementary map.)

The union of this increasing chain of partial elementary maps will itself be a partial elementary map f : Ac → M. Then

Ac≡⌀f(A)f(c) = Af(c)

so that

c≡Af(c)

In particular, tp(f(c)/A) = tp(c/A) = p. Since f mapped into M, we have realized the type p inside M.

QED.


Strongly Homogeneous Models[]

Lemma 3: Let

M0 ≼ M1 ≼ M2 ≼ ⋯

be an elementary chain of models, of length ω, and let N be the union of this chain. Suppose that for each i, Mi+1 is |Mi|+-saturated. Let f : A -> M0 be a partial elementary map for some A ⊆ M0. Then there is an automorphism σ of N extending f.

Proof: We use a back-and-forth argument. First of all, by Tarski-Vaught, each Mi is an elementary substructure of N. So f is a partial elementary map even as a map on subsets of N (rather than M0).

Claim: Let g be a partial elementary map from some subset of Mi to some subset of Mi. Then we can extend g to a partial elementary map whose domain is all of Mi and whose range is contained in Mi+1. Alternatively, we can extend g to a partial elementary map whose domain is contained in Mi+1 and whose range is exactly Mi.

Proof: Let A be the domain of g. Let p(x) be the type of Mi over A. If κ = |Mi|, then Mi + 1 is \kappa^+-saturated. Since |g(A)| < κ+ and x is a tuple of length at most κ+, by the Theorem above we have that the pushforward type g*p (an infinitary type over g(A) ⊂ Mi) is realized in Mi + 1. If D realizes this, then

AMi≡⌀g(A)D.

In particular, the map sending Mi coordinatewise to D is a partial elementary map, extending g. This map has domain exactly Mi and has range contained in Mi + 1.

If we instead wanted the range to be exactly Mi and the domain to be contained in Mi + 1, apply the same argument to g − 1. QEDclaim.

Now build an increasing sequence of partial elementary maps

f0 ⊂ f1 ⊂ f2⋯

with each fi being a partial elementary map from some subset of Mi to some subset of Mi. Do so as follows:

  • f0 is the original map f
  • For n > 0 even, let fn be a partial elementary map from Mn − 1 to a subset of Mn extending fn − 1. This is doable by the Claim.
  • For n > 0 odd, let fn be a partial elementary map from a subset of Mn onto Mn − 1 extending fn − 1. This is doable by the claim.

Let σ be the union of this increasing chain of partial elementary maps on subsets of N. Then σ is itself a partial elementary map. However, the even steps insure that the domain of σ is

$\bigcup\limits_{k = 1}^{\infty}M_{2k - 1} = N$

while the odd steps insure that the range of σ is

$\bigcup\limits_{k = 1}^{\infty}M_{2k} = N$.

In particular, σ is defined on all of N, and is surjective onto N. So σ is in fact an automorphism of N.

QED.

Lemma 4: Let M be a structure. We can find an elementary extension N ≽ M such that every type over M is realized in N, and every partial elementary map from a subset of M to M can be extended to an automorphism of N.

Proof: Build an increasing elementary chain of models

M = M0 ≼ M1 ≼ M2 ≼ ⋯

by inductively choosing Mi+1 to be an |Mi|+-saturated elementary extension of Mi. The union of this chain is an elementary extension of M, by Tarski-Vaught. Every type over M is realized in M1, hence in N. And by Lemma 3, every partial elementary map on subsets of M extends to N.

QED.

Theorem: Let M be a structure, and κ be an infinite cardinal. Then M has an elementary extension which is both κ-saturated and κ-strongly homogeneous.

Proof: As in the proof that κ-saturated models exist, we may replace κ with κ+ and assume that κ is a regular cardinal.

Build an elementary chain {Mα}α < κ as follows.

  • M0 is M
  • Mα + 1 is an elementary extension of Mα such that every type over Mα is realized in Mα + 1 and every partial elementary map from a subset of Mα to Mα extends to an automorphism of Mα + 1. Such an elementary extension of Mα exists by Lemma 4.
  • If α is a limit cardinal, take Mα to be the union of the Mβ for β < α.

Let N be the union of this chain. By Tarski-Vaught, the Mα form an elementary chain and Mα ≼ N for each α.

If A is a subset of N of size less than κ, then by regularity of κ, A ⊂ Mα for some α < κ. Then every type over A is realized in Mα + 1, hence in N. So N is κ-saturated.

For κ-strong homogeneity, let f be a partial elementary map from A to B (subsets of N), with |A| < κ. By regularity of κ, there is some α < κ such that A ∪ B ⊂ Mα. By choice of Mα + 1, there is an automorphism σα + 1 of Mα + 1 extending f.

Recursively define σβ ∈ Aut(Mβ) for α < β ≤ κ (where Mκ = N) as follows:

  • σα + 1 is an automorphism of Mα + 1 extending f. This is doable by choice of Mα + 1.
  • For β > α, σβ + 1 is an automorphism of Mβ + 1 extending the automorphism σβ of Mβ.
  • For λ a limit ordinal, σλ is the union of the increasing chain of σβ for β < λ.

Then the σβ form an increasing chain of automorphisms, each extending f, and σκ is an automorphism of N extending f.

QED.

Saturated Models[]

Recall that M is saturated if M is |M|-saturated.

Theorem: Suppose M and N are two saturated models of the same size, which are elementarily equivalent. Then M and N are isomorphic.

Proof: The type over M over the empty set is consistent with N, because M and N are elementarily equivalent. (The assertion that M and N are elementarily equivalent is equivalent to assertion that the empty map between Ø ⊆ M and Ø ⊆ N is a partial elementary map. Push forward tp(M/Ø) along this map.) Since M is a tuple of length at most |N|, the type of M is realized in N. This means that there is an elementary embedding of M into N. Similarly, we can embed N into M. With a bit more work (a back-and-forth argument) we can obtain an isomorphism between M and N. This is done as follows:

Let κ = |M| = |N|. Choose enumerations {mα}α < κ and {nα}α < κ of M and N, respectively. Build up by induction on α a sequence {fα}α < κ of partial elementary maps from subsets of M to subsets of N. The fα's will form an increasing chain, and fα will have domain and range of cardinality at most ℵ0 + |α|. Proceed as follows:

  • f0 will be the empty map. This is a partial elementary map because M ≡ N.
  • If λ is a limit ordinal, define fλ to be the union of the fα for α < λ. This will be a partial elementary map because its restriction to any finite subset is a partial elementary map. Its domain and range will have size at most |λ|2 = |λ|, because the domain of fα is a union of λ-many sets of size at most |λ|.
  • Given fα, define fα + 1 as follows. Since α < κ, the domain and range of fα have size at most |α| < κ, and therefore must not be all of M or N, respectively. In particular, every 1-type over the range of fα is realized in N. Hence, by Lemma 2, we can extend fα to a partial elementary map on dom(fα) ∪ {mα}. Calling the resulting map gα, we now run the same argument on gα − 1 to extend gα to a partial elementary map whose range includes nα. Let fα + 1 be this new partial elementary map. The domain and range of fα + 1 are bigger than the domain and range of fα by at most two, so the condition on the cardinalities of the domain and range of fα + 1 are satisfied.

Let f be the union of all the fα. This is a partial elementary map from some subset of M to some subset of N. By the construction, mα is in the domain of fα and hence in the domain of f, for every α. In particular, the domain of f contains all of M. Similarly, the range of f contains all of N. As a partial elementary map, f is an injection, so f is a bijection from M to N. Then f is an isomorphism from M to N.

QED.

Corollary: Let T be a complete theory. Then T has at most one saturated model, up to isomorphism, of each cardinality.

Corollary: Let M be a saturated structure. Then M is |M|-strongly homogeneous.

Proof: Let f : A -> B be a partial elementary map between two subsets of M of cardinality less than |M|. Let L be the language/signature of M, and let L(A) be the language obtained by adding a constant ca for every element a ∈ A. There is a tautological way to view M as an L(A)-structure, by interpreting ca. Let M1 denote this expansion of M to the language L(A).

We claim that M1 is saturated. If S ⊆ M is a subset of size less than |M|, then S ∪ A also has size less than |M|. An L-formula over S ∪ A is the same thing as an L(A)-formula over S, so a 1-type over S ∪ A in M is equivalent to a 1-type over S in M1. In particular, the fact that all 1-types over S ∪ A is realized in M implies that all 1-types over S are realized in M1. So M1 is saturated.

We can also make M into an L(A)-structure in another way, by interpreting ca as f(a). Call this expansion M2. By the same argument, M2 is saturated.

Moreover, M1 and M2 are elementarily equivalent, essentially because f is a partial elementary map. Indeed, for any L-formula ϕ(x1, …, xn),

M1 ⊨ ϕ(ca1, …, can) ⇔ M ⊨ ϕ(a1, …, an) ⇔ M ⊨ ϕ(f(a1), …, f(an)) ⇔ M2 ⊨ ϕ(ca1, …, can)

where the middle equivalence follows from the fact that f is an elementary map.

So M1 and M2 are elementarily equivalent saturated structures of the same size (since they have the same underlying set). By the Theorem, they must be isomorphic. Let σ : M1 → M2 be an isomorphism witnessing this. Then σ induces an isomorphism of the reducts to L, which are both M. So σ induces an automorphism of M. Moreover, to be a morphism of L(A)-structures, σ must send the interpretation of ca in M1 to the interpretation of ca in M2. In other words, σ must send a to f(a). So σ extends f.

QED.

Criteria for the Existence of Saturated Models[]

Theorem: Let {Mi : i < ω} be a countable list of structures in some countable language L. Let 𝒰 be a non-principal ultrafilter on ω. Then the ultraproduct $M = \prod\limits_{i < \omega}M_{i}/\mathcal{U}$ is ℵ1-saturated.

Proof: Since the language is countable, any type over a countable subset of M must consist of countably many formulas. So it suffices to show that any countable consistent partial type over M is realized in M. A countable partial type can be enumerated as {ϕi(x; ai) : i < ω}. Each ai can be represented by some element

$(a_{i,1},a_{i,2},\ldots) \in \prod\limits_{j < \omega}M_{j}$,

where each ai, j is a tuple in Mj of the same length and sort as ai in M.

The fact that this partial type is consistent means that for each n,

$M \vDash \exists x:\bigwedge\limits_{i < n}\phi_{i}(x;a_{i})$

In particular, by Łoś's theorem, for "most" values of j,

$M_{j} \vDash \exists x:\bigwedge\limits_{i < n}\phi_{i}(x;a_{i,j})$.

Let

$S_{n} = \{ j:M_{j} \vDash \exists x:\bigwedge\limits_{i < n}\phi_{i}(x;a_{i,j})\}$

Then

S0 ⊇ S1 ⊇ S2 ⊇ S3⋯

and each Sn ∈ 𝒰 (by Łoś's theorem). Let

Tn = Sn ∩ {n, n + 1, n + 2, …},

and let T − 1 = ℕ. Then Tn ∈ 𝒰, because both Sn and {n, n + 1, n + 2, …} are in 𝒰 (the latter because 𝒰 is non-principal). Also, the Tn form a descending sequence:

T − 1 ⊇ T0 ⊇ T1 ⊇ T2 ⊇ ⋯

For each j, let n(j) be the largest n such that j ∈ Tn. A largest such n exists because if j ∈ Tn, then j ≥ n. Choose a singleton cj ∈ Mj as follows:

  • If n(j) = -1, choose cj randomly.
  • Otherwise, j ∈ Tn(j) ⊂ Sn(j), so

$M_{j} \vDash \exists x:\bigwedge\limits_{i < n(j)}\phi_{i}(x;a_{i,j})$

Let cj be an x ∈ Mj witnessing this. So

$M_{j} \vDash \bigwedge\limits_{i < n(j)}\phi_{i}(c_{j};a_{i,j})$

Let c be the class of

(c0, c1, c2, …)

in the ultraproduct M.

Note that if j ∈ Tm + 1, then m < n(j), by definition of n(j). Then by choice of cj, we have

$M_{j} \vDash \bigwedge\limits_{i < n(j)}\phi_{i}(c_{j};a_{i,j})$,

and in particular

Mj ⊨ ϕm(cj; am, j).

Consequently,

{j : Mj ⊨ ϕm(cj; am, j)} ⊇ Tm + 1.

Since Tm + 1 is "big" with respect to the ultraproduct 𝒰, so is the even-bigger set

{j : Mj ⊨ ϕm(cj; am, j)}.

By Łoś's Theorem, it follows that

M ⊨ ϕm(c; am).

This holds for each m, so c satisfies our original partial type {ϕi(x; ai)}. In particular, the type is realized in M.

QED.

Corollary: Assume the continuum hypothesis. Then every consistent theory T in a countable language has a saturated model. Proof: Without loss of generality, T is complete. If the models of T are finite, then they are automatically κ-saturated for all κ, and there is nothing to show. So assume the models of T are infinite. Let M be some countable model. Let 𝒰 be a non-principal ultrafilter on ℕ, and let N = M𝒰 be the corresponding ultrapower. Then by the Theorem, N is ℵ1-saturated. But N is a quotient of Mℕ which has size 2ℵ0. By the continuum hypothesis, this is at most ℵ1, so

|N| ≤ ℵ1

Therefore, N is |N|-saturated. Also, N ⊨ T because N is an ultrapower of a model of T.

QED.

Some variant of these facts works in other cardinalities, though one has to be more careful about the choice of the ultrafilter. In particular, if one assumes the Generalized Continuum Hypothesis, one can show that every structure has a saturated elementary extension. (Somebody who knows about good ultrafilters should verify this.)

Lemma 5: Let M be a structure in a language L. Then there is an elementary extension N ≽ M such that every type over M is realized in N, and |N| ≤ 2|M| + |L|.

Proof: Let N0 be an elementary extension of M in which all types are realized (Lemma 1 above). A type p(x) over M is determined by which L(M) formulas ϕ(x) are true of the variable x. There are only (|M| + |L|)-many L(M)-formulas, so there are at most 2|M| + |L| types over M. For each type over M, choose a realization in N0. Let S be the set of all these realizations. Thus |S| ≤ 2|M| + |L|. By Downward Löwenheim–Skolem, we can find an elementary substructure N ≼ N0 containing S ∪ M, of size |S ∪ M| ≤ 2|M| + |L|. Then N is an elementary extension of M,

|N| ≤ 2|M| + |L|

and every type over M is realized in S, hence in N.

QED.

Theorem: Assume there exists a proper class of strongly inaccessible cardinals. Then every structure M has an elementary extension which is saturated.

Proof: Let κ be an inaccessible cardinal greater than the size of M and greater than the size of the language. Build an increasing elementary chain {Mα}α < κ as follows:

  • M0 = M
  • If λ is a limit ordinal, then

$M_{\lambda} = \bigcup\limits_{\alpha < \lambda}M_{\alpha}$

  • Given Mα, let Mα + 1 be an elementary extension of Mα in which all types over Mα are realized, but |Mα + 1| ≤ 2|Mα| + |T|. Such a model can be found by Lemma 5.

By induction on α, we see that |Mα| < κ. The base case is by choice of κ. The limit ordinal case follows from the regularity of κ. The successor ordinal step follows from the fact that κ is a strong limit cardinal:

|Mα + 1| ≤ 2|Mα| + |T| < κ,

because

|Mα| + |T| < κ

by induction.

Let N be the limit structure. Then N is κ-saturated, by the same arguments used in the theorems above (using the fact that κ is regular). Since N is the union of κ-many structures of size less than κ, N has size κ (or less). Therefore, N is |N|-saturated.

QED.
