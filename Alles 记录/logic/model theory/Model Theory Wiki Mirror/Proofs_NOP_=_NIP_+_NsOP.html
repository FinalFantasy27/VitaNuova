Fix some complete theory T with monster model 𝕌.

A formula ϕ(x; y) has the order property if there exists a1, b1, a2, b2, … in 𝕌 such that i < j ⇔  ⊨ ϕ(ai; bj) for every i, j. T is said to be NOP (or stable) if no formula has the order property.

A formula ϕ(x; y) has the strict order property if there exist b1, b2, … such that ϕ(𝕌; b1) ⊊ ϕ(𝕌; b2) ⊊ ϕ(𝕌; b3) ⊊ ⋯ Taking ai to be in ϕ(𝕌; bi + 1) ∖ ϕ(𝕌; bi), one sees that any formula having the strict order property has the order property. T is said to be NsOP if no formula has the strict order property. Clearly NOP implies NsOP.

Remark. An equivalent condition to NsOP is that T has no interpretable partial order with an infinite chain.

Proof. If (P,  ≤ ) is some interpretable poset, then P is the quotient of some definable pre-order (P′,  ≤ ). The existence of an infinite chain implies that for each n, we can find b1, …, bn such that b1 ≤ ⋯ ≤ bn and bn ≰ bn − 1 ≰ ⋯ ≰ b1. By compactness we can find an infinite sequence b1 ≤ b2 ≤ ⋯ with bi ≱ bi + 1 for each i. Let ϕ(x; y) assert that x, y ∈ P′ and x ≤ y. Then ϕ(𝕌; b1) ⊊ ϕ(𝕌; b2) ⊊ ⋯, so T has the strict order property.

Conversely, suppose that T has the strict order property. Then there is a formula ϕ(x; y) and b1, b2, … with ϕ(𝕌; bi) ⊊ ϕ(𝕌; bi + 1). Let P′ be the sort of y and let ≤ be the pre-order on P′ given by b1 ≤ b2 ⇔ ϕ(𝕌; b1) ≤ ϕ(𝕌; b2). If (P,  ≤ ) is the quotient partial order, then P has an infinite chain. QED

A formula ϕ(x; y) has the independence property if there exist ai for i ∈ ℕ and bS for S ∈ ℕ such that i ∈ S ⇔  ⊨ ϕ(ai; bS) for every i, S. If ϕ(x; y) has the independence property, witnessed by ai and bS, then  ⊨ ϕ(ai; b1, …, j − 1) ⇔ i < j so ϕ has the order property. T is said to be NIP (or dependent) if no formula has the independence property. Clearly NOP implies NIP.

Theorem. T is NOP (stable) if and only if T is both NIP and NsOP.

Proof. We have already noted that NOP implies NIP and NsOP. Conversely, suppose T is NIP and NsOP.

Lemma. Suppose T is NsOP. Let b1, b2, … be a C-indiscernible sequence. Let ϕ(x; y) be a formula. Suppose there is a such that  ⊨ ϕ(a; b2) ∧ ¬ϕ(a; b1) Then there is some a′≡Ca such that  ⊨ ϕ(a′; b1) ∧ ¬ϕ(a′; b2).

Proof. Suppose not. Then tp(a/C) is inconsistent with ϕ(x; b1) ∧ ¬ϕ(x; b2). By compactness, some finite subtype of tp(a/C) is inconsistent with that formula. Therefore there is some c ∈ C and formula ψ(x; z) such that ψ(a; c) holds but ψ(x; c) ∧ ϕ(x; b1) ∧ ¬ϕ(x; b2) is inconsistent. Let ϕ′(x; y, z) be the formula ϕ(x; y) ∧ ψ(x; z). Then ϕ′(x; b1, c) ∧ ¬ϕ′(x; b2, c) is inconsistent. On the other hand, ϕ′(x; b2, c) ∧ ¬ϕ′(x; b1, c) is consistent, being satisfied by a.

This means that ϕ′(𝕌; b1, c) ⊊ ϕ′(𝕌; b2, c). Since b1, b2, … is c-indiscernible, ϕ′(𝕌; bi, c) ⊊ ϕ′(𝕌; bi + 1, c) for each i. So T has the strict order property, a contradiction. QED

Lemma. Suppose T is NsOP. Let {bi}i ∈ ℚ be an indiscernible sequence. Let ϕ(x; y) be a formula. Suppose that S and S′ are subsets of {1, …, n} having the same cardinality. If there is an a such that S = {i ∈ {1, …, n}:⊨ ϕ(a; bi)} then there is a′ such that S′ = {i ∈ {1, …, n}:⊨ ϕ(a′; bi)}

Proof. It suffices to consider the case where the symmetric difference of S and S′ is of the form {j, j + 1}, since we can get between any two subsets of {1, …, n} of the same size via such steps. So j is in one of S and S′, and j + 1 is in the other.

Replacing ϕ with ¬ϕ, we may assume that S ∖ S′ = {j + 1} and S′ ∖ S = {j}. Let C be {b1, …, bj − 1, bj + 2, …, bn}. Note that the sequence bj, bj + 1, bj + 2 − 1/2, bj + 2 − 1/3, bj + 2 − 1/4, bj + 2 − 1/5, … is C-indiscernible. Also, ϕ(a; bj + 1) holds and ϕ(a; bj) does not, because j + 1 ∈ S and j ∉ S. By the previous lemma, we can find a′≡Ca such that ϕ(a′; bj) holds and ϕ(a′; bj + 1) does not hold. If i ∈ {1, …, n} is not one of j or j + 1, then bi ∈ C, so ϕ(a; bi) holds if and only if ϕ(a′; bi) holds, because a′≡Ca. Therefore, {i ∈ {1, …, n}:⊨ ϕ(a′; bi)} = S ∖ {j + 1} ∪ {j} = S′. QED
