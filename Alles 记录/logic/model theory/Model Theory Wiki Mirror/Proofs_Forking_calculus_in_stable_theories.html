Work in a stable theory T with monster model 𝕌.

This article continues where Proofs:Equivalent Definitions of Stability leaves off. We will make use of the following fact from that page:

Fact. Let p(x) be a type over a set A. Let ϕ(x; y) be a formula. Then there is at least one q(x) ∈ Sϕ(𝕌) which is consistent with p(x) and which has finite orbit under Aut(𝕌/A).

Strong Types Have Canonical Extensions[]

If T is stable, so is Teq. To see this, let ϕ(x; y) be a formula in Teq. Then x and y come from sorts of the form X1/E1 and X2/E2, where X1 and X2 are sorts (or products of sorts) in T, and E1 and E2 are definable equivalence relations on X1 and X2. Let πi : Xi → Xi/Ei be the natural projection map, and let ψ(z; w) = ϕ(π1(z); π2(w)). Then ψ(z; w) is (equivalent to) a formula in T, so it does not have the order property. If ϕ(x; y) has the order property, then there exist ai, bi for i < ω such that  ⊨ ϕ(ai; bj) if and only if i < j. Choosing ci ∈ π1 − 1(ai) and di ∈ π2 − 1(bi), we get that  ⊨ ψ(ci; dj) if and only if i < j, a contradiction.

Assume for the rest of this section that T is a stable theory with elimination of imaginaries. (If T is stable without elimination of imaginaries, replace T with Teq.) The assumption that we have elimination of imaginaries will be used mainly in the following way:

Observation 1. Let A be a small algebraically closed set, acl(A) = A. Let X be a definable set. Suppose X has finitely many conjugates under Aut(𝕌/A). Then X is A-definable, and hence has a unique conjugate under Aut(𝕌/A).

Proof. If [X] is a code for X, then [X] has finitely many conjugates over A, hence is in acl(A) = A. Since X is [X]-definable, X is A-definable. QED

Lemma. Let A = acl(A) be a small set. Let p(x) be a complete type over A and let ϕ(x; y) be a formula. Then there is a global ϕ-type which is A-invariant (i.e., Aut(𝕌/A)-invariant), and which is consistent with p(x).

Proof. By the fact at the beginning of this page, we can find a global ϕ-type q which is consistent with p(x) and has finitely many conjugates over A. Now if ψ(y) is the ϕ-definition of q(x), that is, ϕ(x; b) ∈ q(x) ⇔  ⊨ ψ(b) for every q(x), then the 𝕌-definable set ψ(𝕌) has finitely many conjugates over A. By Observation 1, ψ(y) must be A-definable. So q(x) is A-invariant. QED

Recall that if ϕ1(x; y) and ϕ2(x; z) are two formulas, we can find a third formula ψ(x; w) such that the sets of the form ψ(𝕌; c) are precisely the sets of the form ϕ1(𝕌; a) and ϕ2(𝕌; b). Specifically, we take ψ(x; y, z, w′, w″) to be (ϕ1(x; y) ∧ (w′ = w″)) ∨ (ϕ2(x; y) ∧ (w′ ≠ w″)) So ψ acts like ϕ1 if the dummy variables w′ and w″ agree, and acts like ϕ2 if they do not agree.

Iterating this, one sees that given ϕ1(x; y1), …, ϕn(x; yn), we can find ψ(x; w) such that the formulas of the form ψ(𝕌; c) are exactly the formulas of the form ϕi(x; b) for some i and b. Consequently, given a finite set of formulas Δ, we can find a single formula ψ such that Δ-formulas are the same thing as ϕ-formulas, and consequently Δ-types are the same thing as ψ-types.

By the previous lemma, we conclude that if Δ is a finite set of formulas, if A = acl(A), and if p(x) is a complete type over A, then there is a global Δ-type which is consistent with p(x) and A-invariant.

From this, it essentially follows by compactness that types over algebraically closed sets have global invariant extensions:

Lemma. Let A = acl(A) be a small set. Let p(x) be a complete type over A. Then there is a global A-invariant complete type p′(x) extending p(x). The type p′(x) is A-definable.

Proof. Let Σ(x) be the partial type over 𝕌 asserting that for every formula ϕ(x; y), and every b≡Ac, ϕ(x; b) ↔︎ ϕ(x; c) Then p(x) ∪ Σ(x) is consistent. To see this, for each finite set of formulas Δ, let ΣΔ(x) assert that ϕ(x; b) ↔︎ ϕ(x; c) for every ϕ ∈ Δ and every b≡Ac. Then Σ(x) is the union of the ΣΔ(x)’s as Δ ranges over bigger and bigger sets, so it suffices to see that p(x) ∪ ΣΔ(x) is consistent for each Δ. But this follows from the fact that there is a global A-invariant Δ-type which is consistent with p(x).

So p(x) ∪ Σ(x) is consistent. If p′(x) is any complete type extending it, then p′(x) is A-invariant.

By stability, p(x) is definable. For each formula ϕ, the ϕ-definition of p(x) is A-invariant, hence A-definable. So p(x) is A-definable. QED

In fact, the type p′(x) is uniquely determined by p(x). To prove this, we will need a preliminary version of the symmetry of forking.

Recall that if p(x) and q(y) are two global A-invariant types (in any theory), there is a well-defined global A-invariant type p(x) ⊗ q(y) which is characterized by the fact that for any B, the following are equivalent:

  • (a, b) ⊨ (p ⊗ q)|B.
  • b ⊨ q|B and a ⊨ p|Bb.

Theorem 2. Let A be a small set of parameters. Let p(x) and q(y) be two global A-definable types. Then p(x) ⊗ q(y) = q(y) ⊗ p(x).

Proof. If not, there is some formula ϕ(x; y; c) which belongs to p(x) ⊗ q(y) but not to q(y) ⊗ p(x). Let ai, bi be a sequence recursively constructed as follows:

  • bi is a realization of q restricted to Ac ∪ {aj : j < i} ∪ {bj : j < i}
  • ai is a realization of p restricted to Ac ∪ {aj : j < i} ∪ {bj : j ≤ i}.

So essentially the sequence b1, a1, b2, a2, … is a realization of (q ⊗ p ⊗ q ⊗ p ⊗ ⋯)|Ac.

If i < j, then (ai; bj) ⊨ (p ⊗ q)|Ac, so ϕ(ai; bj; c) holds. On the other hand, if i ≥ j, then (bj; ai) ⊨ (q ⊗ p)|Ac, so ϕ(ai; bj; c) does not hold. Therefore ϕ(x; y, z) has the order property, a contradiction. QED

Corollary 3. Let A = acl(A). Then each complete type over A has a unique global A-invariant extension.

Proof. Let p and p′ be two global A-invariant types such that p|A = p′|A. Suppose p ≠ p′. Then there is some formula ϕ(x; b) which is in p but not in p′. Let a realize p|Ab and a′ realize p|Ab′. Then ϕ(a; b) holds and ϕ(a′; b) does not, so (a, b)≢A(a′, b).

Let q be a global A-invariant type extending tp(b/A). Note that (a, b) ⊨ (p ⊗ q)|A and (a′, b) ⊨ (p′ ⊗ q)|A. By Lemma 2, (b, a) ⊨ (q ⊗ p)|A and (b, a′) ⊨ (q ⊗ p′)|A. Now (q ⊗ p)|A depends on p only through p|A, so we should in fact have (q ⊗ p)|A = (q ⊗ p′)|A, contradicting (a, b)≢A(a′, b).

More precisely, let σ be an automorphism over A moving a′ to a. Since b ⊨ q|Aa′, and q is fixed by σ, we get that σ(b) ⊨ q|Aa. As b ⊨ q|Aa, we get that σ(b)≡Aab, or equivalently, (a, b)≡A(a, σ(b)) = σ((a′, b)). So (a, b)≡A(a′, b), a contradiction. QED

Although we have proven Corollary 3 for finitary types, it holds equally well for infinitary types, by an easy exercise. One way to see this is to verify that the above proof works for infinitary types. (Infinitary ϕ-types turn out to be equivalent to finitary ϕ-types, because a single formula ϕ can only reference finitely many of the variables in an infinite tuple.) Alternatively, one can deduce Corollary 3 for infinitary types from Corollary 3 for finitary types, by using the fact that $S_{I}(A) = \lim\limits_{\rightarrow}S_{I_{0}}(A)$ where the limit ranges over finite subsets I0 of I, and SI(A) denotes the set of complete types over A of tuples ⟨ai⟩i ∈ I indexed by I.

Forking[]

Suppose T is stable, but perhaps does not eliminate imaginaries. If C is a small subset of the monster 𝕌, or of 𝕌eq, then acleq(C) denotes the algebraic closure of C within 𝕌eq.

If a is a tuple, the strong type of a over C, denoted stp(a/C) is by definition just tp(a/acleq(C)). From the previous section, we know that stp(a/C) has a unique extension to a global acleq(C)-invariant type. By a slight abuse of notation, denote this stp(a/C)|𝕌. We are thus using the symbol | to denote extension, not restriction. Also, if B is any subset of 𝕌eq, we let stp(a/C)|B denote the restriction of stp(a/C)|𝕌 to B.

Fix some small base set C. Let C′ = acleq(C). If a and b are (possibly infinite) tuples, let a↓Cb indicate that there exist global C′-definable types p and q such that (a, b) ⊨ (p ⊗ q)|C′. Then a ⊨ p|C′ and b ⊨ q|C′. Since types over C′ have unique C′-definable extensions, p must be tp(a/C′)|𝕌, q must be tp(b/C′)|𝕌, and p ⊗ q must be tp(ab/C′)|𝕌. So a↓Cb ⇔ (stp(a/C)|𝕌) ⊗ (stp(b/C)|𝕌) = stp(ab/C)|𝕌.

If p = stp(a/C)|𝕌 and q = stp(b/C)|𝕌, then a↓Cb is equivalent to stp(ab/C)|𝕌 = p ⊗ q. Because types over C′ are in bijective correspondence with global C′-definable types, this is equivalent to ab ⊨ p ⊗ q|C′. From the definition of p ⊗ q, this is the same as saying that

  1. b ⊨ q|C′
  2. a ⊨ p|C′b.

But (1) is automatic, since q = tp(b/C′)|𝕌. So a↓Cb is equivalent to a ⊨ p|C′b, or equivalently, a ⊨ tp(a/C′)|C′b.

At this point, we have at least two equivalent definitions for a↓Cb:

  • (a, b) ⊨ p ⊗ q|C′ for some C′-definable types p and q.
  • a ⊨ stp(a/C)|C′b.

where C′ = acleq(C).

Lemma.

(a)
a↓Cb ⇔ b↓Ca. This fact is called symmetry.
(b)
a↓Cb depends only on the underlying sets of a and b, and not on the orderings of the tuples a and b. So ↓ is really a relation on sets, not tuples.
(c)
If A′ ⊆ A and B′ ⊆ B, then A↓CB implies A′↓CB′. This fact is called monotonicity.
(d)
A↓CB holds if and only if A′↓CB′ holds for all finite subsets A′ of A and B′ of B. This fact is called finite character.
(e)
If A↓CB and D is another set, then we can find A′≡BCA such that A′↓CBD. This is called right extension.
(f)
If σ is an automorphism of 𝕌, then A↓CB ⇔ σ(A)↓σ(C)σ(B). This is called automorphism invariance. Equivalently, if A′B′C′≡⌀ABC, then A↓CB ⇔ A′↓C′B′.
(g)
If A↓CB, then A↓Cacleq(BC).

Proof.

(a)
This comes from Theorem 2 above, and the first definition of ↓.
(b)
The fact that a↓Cb depends only on b as a set, not as a tuple, is clear from the second of the two definitions of ↓ given above. Then by symmetry, it only depends on a as a set.
(c)
If a↓CB holds and B′ ⊂ B, then a realizes p|C′B, where p = tp(a/C′)|𝕌. Clearly the tuple a also realizes the smaller type p|C′B′, so a↓CB′. This gives monotonicity on the right: A↓CB ⇒ A↓CB′ if B′ ⊂ B. By symmetry we also get monotonicity on the left.
(d)
By symmetry, it suffices to show that a↓CB holds if and only if a↓CB′ holds for all finite subsets B′ ⊂ B. Let p = tp(a/C′)|𝕌. By the second definition of ↓, a↓CB′ is equivalent to a ⊨ p|C′B′ for all B′. But a realizes p|C′B if and only if a realizes the restriction of that type to all finite subsets. So a realizes p|C′B if and only if a realizes p|C′B′ for all finite B′ ⊂ B.
(e)
Suppose a↓CB and D is some set. Let p = tp(a/C′)|𝕌. Let a′ realizes p|C′BD. Then tp(a′/C′) = tp(a/C′), so p is also tp(a′/C′)|𝕌. Therefore a′↓CBD is equivalent to a′ ⊨ p|C′BD, which is true. To see that a′≡BCa, note that because a↓CB, we know that a realizes p|C′B. Thus a′ and a both realize p|CB, and therefore a≡BCa′.
(f)
This is clear from the definitions, which broke no symmetries of the monster.
(g)
By right extension, we can find A′≡BCA such that A′↓Cacleq(BC). Let σ be an automorphism fixing BC and sending A′ back to A. By automorphism invariance, A↓Cσ(acleq(BC)). But σ fixes BC, so it can at most permute the members of acleq(BC). In particular, σ(acleq(BC)) is the same underlying set as acleq(BC), so by part (a), A↓Cacleq(BC).

QED

By monotonicity and part (g), we have the following equivalence A↓CB ⇔ A↓Cacleq(BC). By the second definition of ↓, we conclude A↓CB ⇔ A ⊨ p|acleq(BC), where p is stp(A/C)|𝕌. The type p is acleq(C)-definable, hence acleq(BC)-definable. By stability, types over acleq(BC) have unique global acleq(BC)-definable extensions, so A ⊨ p|acleq(BC) is equivalent to stp(A/BC)|𝕌 = p.

In summary, A↓CB is equivalent to the following: stp(A/C)|𝕌 = stp(A/BC)|𝕌. This gives a third definition of ↓.

Lemma. Suppose C1 ⊂ C2 ⊂ C3 and A is a set. Then A↓C1C3 if and only if A↓C1C2 and A↓C2C3. This fact is called transitivity.

Proof. First suppose that A↓C1C2 and A↓C2C3. By the third definition of ↓, we see that stp(A/C1)|𝕌 = stp(A/C2)|𝕌 = stp(A/C3)|𝕌. By transitivity of equality, stp(A/C1)|𝕌 = stp(A/C3)|𝕌, which means A↓C1C3 by the third definition of ↓.

Conversely, suppose that A↓C1C3. By right monotonicity, A↓C1C2. Then stp(A/C3)|𝕌 = stp(A/C1)|𝕌 = stp(A/C2)|𝕌. By transitivity of equality, stp(A/C2)|𝕌 = stp(A/C3)|𝕌, which means A↓C2C3 by the third definition of ↓. QED

We now prove the remaining essential facts for the forking calculus:

Lemma.

(1)
A↓BB always holds.
(2)
A↓BA holds if and only if A ⊂ acleq(B).
(3)
Suppose C = acleq(C), or more generally, acleq(C) = dcleq(C). (In particular, this holds if C is a model). If A↓CB and A′↓CB, then A′≡CA implies A′≡BCA.
(4)
There is some cardinal κ = κ(T) such that if a is a finite tuple and C is a small set, then there exists C′ ⊂ C with |C′| ≤ κ and a↓C′C. This is called local character.

Proof.

(1)
By the third definition of ↓, this amounts to showing that stp(A/B)|𝕌 equals itself.
(2)
Suppose A ⊂ acleq(B). By (1), A↓BB. By the lemma above, A↓Bacleq(AB). But A ⊂ acleq(AB), so by monotonicity on the right, A↓BA. Conversely, suppose A↓BA. We want to show that A ⊂ acleq(B). By finite character, we may assume that A is a finite tuple a. Now stp(a/B)|𝕌 = stp(a/Ba)|𝕌. But the constant type x = a is clearly the unique Ba-definable type extending stp(a/Ba), so stp(a/B)|𝕌 must be the constant type x = a. In order for this to be acleq(B)-invariant, the element a must be fixed by every automorphism fixing acleq(B). This forces a to be in acleq(B).
(3)
Suppose A↓CB and A′↓CB and A′≡CA. Because C is algebraically closed, strong types over C are the same as types over C. So stp(A/BC)|𝕌 = tp(A/C)|𝕌 = tp(A′/C)|𝕌 = stp(A′/BC)|𝕌. This ensures that stp(A/BC) = stp(A′/BC), so in particular tp(A/BC) = tp(A′/BC).
(4)
We can take κ to be |T|. Let p = stp(a/C)|𝕌. Then p is acleq(C)-definable. The definition of p consists in a map assigning to each formula ϕ an acleq(C)-formula dpϕ. There are only |T| formulas ϕ, so there must be some subset C′ ⊂ C of size at most |T| such that dpϕ is over acleq(C′) for every ϕ. Thus p is acleq(C′)-definable. Now stp(a/C′) is a restriction of stp(a/C) which is a restriction of p. Since p is acleq(C′)-definable, p must in fact be stp(a/C′)|𝕌. Then stp(a/C′)|𝕌 = p = stp(a/C)|𝕌, so a↓C′C.

QED
