Fix some theory T. Let κ be a cardinal. An ict pattern of depth κ is a collection of formulas ⟨ϕα(x; y)⟩α < κ and constants bα, i for α < κ and i < ω, such that for every η : κ → ω, the following collection of formulas is consistent: $\bigwedge\limits_{\alpha < \kappa}\phi_{\alpha}(x;b_{\alpha,\eta(\alpha)}) \land \bigwedge\limits_{\alpha < \kappa,i \neq \eta(\alpha)}\neg\phi_{\alpha}(x;b_{\alpha,i}).$ So we have an array of formulas, with κ rows and ω columns, each row being uniform, and for every vertical path through the array, there is an element which satisfies exactly those formulas along the path, and no others.

More generally, if Σ(x) is a partial type over some parameters, then an ict pattern of depth κ in Σ(x) is an array as above, such that for each η : κ → ω, $\Sigma(x) \land \bigwedge\limits_{\alpha < \kappa}\phi_{\alpha}(x;b_{\alpha,\eta(\alpha)}) \land \bigwedge\limits_{\alpha < \kappa,i \neq \eta(\alpha)}\neg\phi_{\alpha}(x;b_{\alpha,i})$ is consistent.

Given an ict pattern, we can always extract another ict pattern using the same formulas, but with the ⟨bα, i⟩ mutually indiscernible.

Shelah defines κict of the theory T to be the supremum of the depths of ict patterns, or ∞ if there exist ict patterns of unbounded depth. It turns out that κict < ∞ if and only if T is NIP.

A theory is said to be strongly dependent if there are no ict patterns of depth ℵ0. The maximum depth of an ict pattern in a type Σ(x) is the dp-rank of Σ(x), or some variant thereof. ::: ::: :::
