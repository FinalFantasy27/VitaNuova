According to the substitutional interpretation, ‘∃x φ(x)’ is true if and only if ‘φ(t)’ is true, for some singular term ‘t’ of the language. When the quantifiers are given a substitutional interpretation. If the language has empty singular terms, for example, a theory that contains (a symbolic translation of) the sentence ‘Sherlock Holmes is a fictional detective’ would not be ontologically committed to fictional objects.
